## 一、这生意是什么

帮不懂技术的人安装配置AI工具（OpenClaw/Claude Code/WorkBuddy等），收取服务费。

**为什么有人愿意付钱**：
- 安装涉及命令行、环境变量、Git、Python依赖——普通人根本不会
- AI焦虑驱动："别人都在用AI提效，我不会装"
- 愿意花几百块省掉几小时的折腾

**真实收入参考（2026年）**：
| 服务 | 价格 |
|------|------|
| 远程基础安装 | 100-300元/次 |
| 上门安装 | 300-600元/次（北上广深） |
| 安装+定制工作流 | 500-1000元 |
| 卸载服务（新趋势） | 远程50-100元，上门299元 |
| 付费社群/教程 | 9.9-199元/人 |

## 二、你会装什么

以你已有的能力（会用Claude Code），可以直接提供服务：

| 工具 | 难度 | 用户需求 |
|------|------|----------|
| Claude Code（命令行版） | ⭐⭐ | 需要Node.js环境、Anthropic账号、API Key配置 |
| Claude Code（IDE插件版） | ⭐ | VSCode/Cursor安装插件即可 |
| OpenClaw（龙虾） | ⭐⭐⭐ | 需要Python、Git、Docker、多个API Key |
| Stable Diffusion | ⭐⭐⭐ | 需要Python、CUDA、模型下载（动辄几十G） |
| AI中转站/API代理 | ⭐⭐⭐ | 服务器搭建、Nginx、SSL、限流配置 |

**建议起步**：先做Claude Code代安装（你最熟悉），再扩展到OpenClaw。

### 接单中：沟通话术

**第一步：确认需求**
```
"你好，请问你需要安装哪个工具？Claude Code（编程辅助）还是OpenClaw（全能助手）？"
"是什么操作系统？Windows 10/11 还是 Mac？"
"电脑配置大概是什么样的？（内存/硬盘空间）"
```

**第二步：报价**
```
"远程安装+环境配置+30分钟使用教学，一共XX元。"
"如果你还需要定制化的工作流（比如自动发小红书/自动整理文件），再加XX元。"
```

**第三步：开始服务前确认**
```
"安装前请确认：
1. 网络通畅（最好连网线，别用热点）
2. 关闭杀毒软件（会误拦截）
3. 已经准备好Anthropic账号和API Key
4. 安装ToDesk/向日葵远程软件（我来远程操作）"
```

### 交付时：检查清单

- [ ] 工具能正常启动
- [ ] 基础功能测试通过（发一条消息看回复）
- [ ] 用户自己操作一遍
- [ ] 录屏文件发给用户（已录好的安装回放）
- [ ] 告知常见问题和解决方法
- [ ] 留下联系方式（7天售后）

### 交付后：追销

- "装完后如果想定制工作流（比如自动帮你整理公众号文章/分析Excel），可以找我"
- "我这有个AI使用者交流群，里面有提示词模板和工作流分享，XX元进群"
- 朋友圈/小红书持续发"客户反馈"和"使用技巧"

- 大部分人选中档，但你真正想卖的是进阶版

## 六、常见问题与回答

**Q：装完会不会有安全隐患？**
A：全程录屏，只装开源工具，不碰你任何私人文件。装完你可以改所有密码。

**Q：装完我不会用怎么办？**
A：包含30分钟教学，教你最常用的3个功能。还有录屏回放可以反复看。

**Q：能用多久？会不会过期？**
A：软件本身是开源免费的，不会过期。API费用是你自己付给Anthropic的，跟我没关系。

**Q：装失败了怎么办？**
A：装不好全额退款，你没有任何风险。

## 七、避坑清单

| 坑 | 应对 |
|----|------|
| 客户电脑太旧/32位系统 | 接单前先确认配置，不满足条件的婉拒 |
| 客户网络太差（下载失败） | 提前告知需要稳定网络 |
| 客户装完反手一个退款 | 闲鱼交易走平台，录屏留证据 |
| 被同行恶意低价卷 | 不做纯安装，绑定教学+售后做差异化 |
| 客户要你帮注册账号 | 只教方法，不代注册（涉及实名/手机号） |
| API中转站法律风险 | 不碰API转售，违反Anthropic ToS |

**核心逻辑**：安装是入口，定制才是利润。

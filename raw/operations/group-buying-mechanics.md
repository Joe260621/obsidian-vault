# 活动运营-拼团裂变玩法

- **Source**: 幕布笔记 → Inbox/
- **Collected**: 2026-06-20

## 多人拼团抽奖
- 规则：可快速成团、多产品选择、流量聚焦
- 奖励模式：体验模式（会员）、额外奖励（连中3次获10次不中奖券）

## 中奖者
- 产品4选1送
- 等价基金换购
- 5%的基金释放
- 分红

## 未中奖者
- 本金返还
- 5%广告费
- 5%的基金释放（中奖才有）

## 成本预估案例
以10人成团，3人中奖，7人未中为例（产品成本30元，售价100元，分销一次19元）
- 支出：30×3 = 90元
- 中奖者释放基金：5×3 = 15元
- 广告费：5×7 = 35元
- 未中奖者释放基金：5×7 = 35元（需已中过奖）
- 分销：19×10 = 190元
- 活动总成本：365元

收入：100×10 = 1000元
毛利：1000 - 365 = 635元

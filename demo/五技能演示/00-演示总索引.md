# Obsidian × Claude 五技能演示 🎯

用这5个技能生成的演示文件，直接在 Obsidian 中打开体验 ↓

## 1. 📝 obsidian-markdown
**文件：** [[01-求职分析笔记]]
**看什么：**
- 顶部 frontmatter（标题/标签/别名/状态）
- `[[双链]]` 链接到其他笔记
- `> [!tip]` 和 `> [!warning]` Callout 框
- `%%注释%%` 隐藏文字（仅编辑模式可见）
- `==高亮==` 文本
- Mermaid 图表（思维导图）
- `$n=38$` 行内 LaTeX
- `^block-id` 块引用锚点

## 2. 🧠 json-canvas
**文件：** [[02-求职技能树.canvas]]
**看什么：**
- 在 Obsidian 中打开 → 可视化思维导图
- 6 种颜色的技能节点
- 带标签的连线
- Group 分组框："差异化标签"
- Link 节点链接到 GitHub

## 3. 🗄️ obsidian-bases
**文件：** [[03-投递追踪.base]]
**看什么：**
- Obsidian 中打开 → 自动渲染为数据库视图
- 两个视图："全部投递" 和 "待跟进"
- 公式自动计算等待天数
- 按状态分组（进行中/已投递/已拒绝）
- 可嵌入到笔记：`![[03-投递追踪.base#待跟进]]`

## 4. 🌐 defuddle
**文件：** [[04-JSON-Canvas规范剪藏]]
**看什么：**
- 去广告、去导航栏的纯净正文
- jsoncanvas.org 规范文档的 Markdown 版
- 对比原始网页 vs 剪藏后的干净内容

## 5. 💻 obsidian-cli
**文件：** [[05-Obsidian-CLI命令参考]]
**看什么：**
- 需要 Obsidian 桌面版运行中 + 开启命令行支持
- 用终端直接创建/搜索/管理笔记
- 写插件时的调试命令

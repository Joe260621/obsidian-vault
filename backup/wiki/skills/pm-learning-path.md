## 定位说明

> ⚠️ **这是长期发展方向，不是当前投递目标。** 当前主攻用户运营/私域运营岗位，AI和产品能力作为面试中的差异化加分项。入职后在实际业务中积累产品思维，再考虑内部转岗或跳槽。

---

## 为什么有这个方向

探迹科技（AI独角兽，估值95亿，红杉/阿里投资）的 AI Agent 产品经理岗位（15-30K·14薪）提供了一个可参考的目标画像：

| JD 要求 | 当前状态 | 差距 |
|---------|---------|------|
| 企业微信生态经验 | ✅ 6年私域/企微实操 | 强匹配 |
| 私域运营场景理解 | ✅ 百万人级用户运营 | 强匹配 |
| LLM/ChatGPT应用 | ✅ 熟练使用 + 搭建过AI系统 | 强匹配 |
| 数据指标体系 | ✅ SQL + BI 数据分析 | 中匹配 |
| 产品设计经验 | 🔴 PRD/Figma/产品思维已学 | 需实战 |
| RAG/Tool Calling | 🔴 已系统性学习+跑过Demo | 需深度 |

**核心优势：会写代码、懂AI、做过运营——比纯产品经理更懂技术落地，比纯技术更懂用户场景。**

## 已完成的准备

### 模块1：产品经理基本功
- 产品思维：痛点→根因→方案→价值分析框架
- PRD文档：以希尔顿企微AI客服为案例，输出完整PRD（用户故事/功能需求/非功能需求/验收标准）
- Figma原型：完成手机端对话界面原型

### 模块2：AI Agent核心技术
- RAG（检索增强生成）：动手搭建希尔顿76款产品知识库
- Tool Calling（工具调用）：实现AI自动查价格/查库存
- ReAct（思考循环）：理解多步骤任务推理逻辑
- 面试应答：已准备RAG/Tool Calling/ReAct技术追问的应答

### 实践项目：希尔顿企微AI客服
- 需求分析 → PRD → Figma原型 → AI Demo → 架构设计
- 3天独立完成全流程
- 详见[希尔顿企微AI客服](hilton-ai-customer-service.md)

## 待补充的能力

| 能力 | 重要度 | 学习方式 |
|------|-------|---------|
| 产品数据分析 | 高 | 在工作中学——运营岗天然接触数据 |
| 用户研究方法 | 中 | 实际工作中观察和学习PM怎么做 |
| 跨部门协作 | 高 | 入职后主动参与需求评审和项目沟通 |
| 技术方案评审 | 低 | 先做运营，再逐渐参与技术讨论 |

## 转型路径（假设）

```
当前: 用户运营/私域运营岗位（主攻）
  ↓ 入职后6-12个月
在公司内部主动承担跨部门项目、参与产品需求讨论
  ↓
积累产品思维 + 技术深度 + 项目案例
  ↓ 12-24个月后
内部转岗产品经理 或 跳槽AI产品经理岗位
```

**短期（求职阶段）：** AI+运营能力作为面试差异化武器，不主动提"想转产品"
**中期（入职后）：** 在运营岗用产品思维做事，建立和产品团队的合作关系
**长期（1-2年）：** 有足够的作品和深度后，正式转向产品方向

## 一、法理学

### 法的作用

| 作用类型 | 对象 | 说明 |
|----------|------|------|
| **预测作用** | 人们相互的行为 | 预判他人行为和后果 |
| **评价作用** | 他人行为 | 评价合法还是违法 |
| **指引作用** | 本人行为 | 对自己的行为具有引导作用 |
| **教育作用** | 一般人行为 | 通过实施产生示范或示警 |

## 三、行政法

### 1. 行政处罚

- **种类**：警告、罚款、没收、吊销（工商机构）、拘留（公安机构）
- **实施机关**：行政机关、法律法规授权的事业单位

### 2. 行政许可（信赖保护原则）

- **特征**：依申请、授益性、要式（非口头、书面）行为
- **分类**：排他性 / 非排他性 / 独立 / 附文件
- **法定分类**：
  - 普通许可
  - 特许（数量限制）
  - 认可（资格资质）
  - 核准（物）
  - 登记（主体资格）
- ⚠️ **部委规章、省级以下政府规章不可设定行政许可**
- **实施机关**：行政机关、法律法规授权管理公共事务的行政机关
- **听证**：依职权 / 依申请，20日内组织，7天前告知，作笔录

### 3. 行政强制

- **行政强制措施**：人身自由、财务暂时限制，**不得委托**
- **行政强制执行**（有合法依据）：
  - 直接强制：扣押、划拨、拍卖、查封场所
  - 间接强制：加处罚款、滞纳金、代履行

### 4. 其他行政行为

| 行为类型 | 要点 |
|----------|------|
| 行政裁决 | 特定民事纠纷 |
| 行政调解 | 不具备可诉性、无强制性 |
| 行政仲裁 | 劳动仲裁，主体为仲裁委员会 |
| 行政指导 | — |
| 行政确认 | 工伤认定书、交通事故责任认定书、结婚证、房屋登记证、技术鉴定书 |
| 行政征收 | 税、费、财物所有权 |
| 行政征用 | 使用权 + 补偿 |
| 行政给付 | — |
| 行政合同 | 实现行政职能（政府采购、土地出让、行政征用合同） |
| 行政事实行为 | 不产生法律约束力 |

### 5. 行政复议 vs 行政诉讼

| | 行政复议 | 行政诉讼 |
|------|----------|----------|
| 审查内容 | 合法性 + **合理性** | 合法性 |
| 申请期限 | 60天内 | 6个月内 |
| 受理期限 | 5天内 | 7天内 |
| 决定期限 | 60天（+30天延长） | 6个月 |
| 原则 | **不调解**原则 | 不服复议15天内起诉；简易程序45日 |

## 四、民法

### 1. 基本原则

调整对象：自然人、法人（独立享有）和非法人组织（个人独资企业、合伙企业等）之间的**人身关系**和**财产关系**

**原则**：平等、自愿、公平、诚实信用、**公序良俗**、绿色原则

### 2. 代理

- **表见代理**：本质无权代理，但产生有权代理的后果

### 3. 物权

| 类型 | 内容 |
|------|------|
| 所有权 | 先占、**善意取得**（解决所有权问题，不解决合同效力问题）、拾得遗失权、埋葬物、添附 |
| 用益物权 | 土地承包经营权、宅基地使用权、建设用地使用权、**地役权**（利用他人不动产提高自己不动产权益）、**居住权**（不得转让、继承或出租） |
| 担保物权 | 抵押权、质押权（**转移占有**）、**留置权** |

- **物权变动生效**：动产 → 交付；不动产 → 登记
- **物权 > 债权**

#### 业主的建筑物区分所有权
- 建筑区划内的道路、绿地（城镇公共例外）、物业服务用房、外墙、停车位
- **成员权**：涉钱涉房问题 → 总面积 **2/3** 以上业主 **且** 总人数 **2/3** 以上业主同意；其他过半即可

### 4. 债权

- **产生原因**：合同、单方允诺、侵权行为、**无因管理**、不当得利
- **无因管理**：无法定或约定义务，为避免他人利益受损而管理；有及时通知义务；有为他人谋利益的意思；具有请求偿还必要费用的权利

### 5. 合同

- **本质**：内心意思一致的合法民事法律行为 + 外部表示行为
- **形式**：口头、书面、行为和沉默
- **分类**：单务合同（如赠与）、双务合同
- **相对性**：合同只在当事人之间产生效力

#### 要约与承诺
- **要约邀请**（不构成要约）：寄送价目表、拍卖公告、招标公告、招股说明书、商业广告
- **合同成立地点**：签名盖章的地方

#### 合同效力
- **效力待定**：主体不合格（如未成年、无权代理人）
- **合同无效**：保证人为非国家机关、学校、医院等事业单位时

#### 合同担保与保全
- **定金**：不超过债权 20%，失约**双倍**罚则（订金则不用双倍）
- **代位权**：甲→乙（不还钱）→ 甲直接向丙（乙的债务人）请求还钱
- **撤销权**

### 6. 人身权

| 类型 | 内容 |
|------|------|
| **人格权** | 生命权、身体权、健康权、姓名权、肖像权（侵权要件：未经同意 + 以营利为目的；声音属于肖像权） |
| **身份权** | 荣誉权、监护权、亲权、配偶权、亲属权 |

> ⚠️ 法人有**名称权**，但**无生命权、肖像权**

- 侵害人身权益致严重精神损害 → 可请求**精神损害赔偿**

### 7. 知识产权

**特征**：专有性、地域性、时间性；包括 **财产权 + 人身权**

#### 著作权
- **保护条件**：独创性、可复制性
- **不受保护**：禁止出版内容、法律法规、新闻、历法、表格和公式
- **权利内容**：
  - 著作人身权：发表、署名、修改、保护作品完整
  - 著作财产权：发行、展览
- **保护起点**：创作完成开始
- **保护期**：
  - 自然人：作者终身 + 死后 **50 年**
  - 法人：首次发表 → 第 50 年的 12 月 31 日
- **限制**：
  - 合理使用：不经许可、不给报酬（如个人学习研究、图书馆、改成盲文）
  - **法定许可**：不经许可，**支付使用费**（如报刊转载、编写出版教科书）

#### 专利权
- **类型与期限**：
  - 发明 → **20 年**
  - 实用新型、外观设计 → **10 年**
- **不授予专利**：科学发现、智力活动规则和方法、疾病诊断和治疗方法（但**药品可以**）、动植物品种（但**生产培育方法可以**）、原子核变换方法获得的物质、平面印刷品图案色彩标识设计
- **申请日**：专利部门收到之日；邮寄为寄出邮戳日

### 8. 婚姻法

- **基本原则**：双方自愿、一夫一妻、法定年龄
- **无效/可撤销婚姻**：从"开始时"无效（**自始原则**）
- **准予离婚条件**：
  1. 重婚或与人同居
  2. 家庭暴力或虐待、遗弃成员
  3. 赌博、吸毒恶习
  4. **感情不合分居 2 年**
  5. 其他导致感情破裂的情形

#### 财产归属
- **个人财产**：婚前财产、身体获得的医疗补助费、受赠与的财产、一方专用生活品

### 9. 继承法

| 顺位 | 人员 |
|------|------|
| **第一顺位** | 配偶、子女、父母（含尽了主要赡养义务的公婆、岳父母） |
| **第二顺位** | 兄弟姐妹、祖父母、外祖父母 |

**遗嘱方式**：自书、代书（两见证人）、录音（两见证人）、**口头**（危急 + 两见证人）、公证

### 10. 侵权责任

- **构成要件**：行为、结果、因果关系、过错
- **归责原则**：

| 原则 | 说明 | 示例 |
|------|------|------|
| 过错责任 | 无过错即无责任 | — |
| **推定过错** | 行为人致害，不能证明无过错则推定担责 | 动物园动物致害、物件致人损害 |
| **无过错责任** | 不论有无过错都担责 | 监护人责任、职务侵权、饲养动物伤人 |

- **机动车**：使用人承担赔偿；转让并交付但未办理转移登记 → 受让人承担

### 11. 诉讼时效

- **期限**：**3 年**
- **效果**：丧失**胜诉权**，不丧失起诉权和实体权利
- **适用范围**：债权请求权
- **除斥期间**：实体权利的存续期间（不同于诉讼时效）
- **不适用情况**：停止侵害、消除危险；不动产物权返还；请求支付抚养费、赡养费

## 五、刑法

### 1. 基本原则：罪刑法定

- 成文法定（排斥习惯法）
- 事前法定（禁止事后法）
- 严格法定（禁止类推解释）

### 2. 管辖

| 类型 | 适用情形 |
|------|----------|
| **属地管辖** | 中国境内犯罪 |
| **属人管辖** | 中国人在国外犯罪 |
| **保护管辖** | 外国人在国外对中国人犯罪 |
| **普遍管辖** | 我国参加的国际条约规定的犯罪 |

### 3. 犯罪构成

#### 犯罪客体
- 一般客体 → 所有犯罪侵犯
- **同类客体** → 某一类犯罪侵犯（**犯罪分类的依据**）
- 直接客体 → 某一具体犯罪侵犯
- 犯罪对象：犯罪行为直接作用的人或物

#### 犯罪主体（刑事责任年龄）

| 年龄段 | 责任类型 | 说明 |
|--------|----------|------|
| < 14 周岁 | 无责任 | — |
| **14-16 周岁** | 相对责任 | 故意杀人、重伤、强奸、抢劫、**毒**等 8 种 |
| ≥ 16 周岁 | 完全责任 | — |

- **从轻/减轻**：满 14 不满 18 周岁；满 75 周岁过失犯罪
- **可以从轻/减轻**：满 75 周岁故意犯罪
- **可以**从轻/减轻/**免除**：又聋又哑或盲人犯罪

#### 犯罪主观方面

| 类型 | 特征 |
|------|------|
| **直接故意** | 明知结果 + 希望发生（积极追求） |
| **间接故意** | 明知结果 + 放任发生（无所谓） |
| **过于自信过失** | 应当预见、已经预见、轻信能避免 |
| **疏忽大意过失** | 应当预见而没有预见 |

### 4. 排除犯罪行为

#### 正当防卫
- 条件：有不法侵害 → 侵害未结束 → 对侵害者本人 → 没超过必要限度
- **防卫过当**：**应当**减轻或免除处罚
- ⚠️ 行凶、杀人、抢劫、强奸、绑架等危及人身安全的暴力犯罪 → **无过当防卫**

#### 紧急避险
- 不适用于职务、业务负有特定责任的人
- **应当**减轻或免除

### 5. 犯罪形态

- **犯罪既遂**：结果犯（盗窃、诈骗、杀人）、行为犯、危险犯
- ⚠️ 既遂后不会转化为未遂、中止

### 6. 共同犯罪

| 角色 | 处罚 |
|------|------|
| 主犯 | 按**全部罪刑**处罚 |
| 从犯 | **应当**减轻或免除 |
| 胁从犯 | **应当**减轻或免除 |
| 教唆犯 | 教唆不满 18 周岁 → **应当从重处罚** |

### 7. 刑罚

#### 主刑

| 刑种 | 期限 | 特点 |
|------|------|------|
| **管制** | 3 月 - 2 年（数罪并罚 ≤ 3 年） | 不予关押，限制自由；羁押 1 日抵刑期 2 日 |
| **拘役** | 1 - 6 月（数罪并罚 ≤ 1 年） | 短期关押；酌量报酬；每月回家 1 次 |
| 有期徒刑 | — | — |
| 无期徒刑 | — | — |
| **死刑** | — | 罪行极其严重；不满 18 周岁 / 审判时怀孕妇女 / 审判时满 75 周岁（特别残忍手段致死除外）→ 不适用；**死刑由最高人民法院核准**；死缓可由高级人民法院核准 |

#### 附加刑
- 罚金、剥夺政治权利（选举权、言论出版游行示威权、不能当官）

### 8. 量刑制度

- **一般累犯**：前后两罪都是故意犯罪 + 都是有期徒刑 + **5 年内**犯第 2 次
- **特殊累犯**：危害国家安全、恐怖活动、黑社会性质
- ⚠️ 累犯 → **应当从重处罚**，**不适用缓刑**

#### 数罪并罚
- **吸收**：有期 + 拘役 = 有期
- **并科**：附加刑 / 管制 + 有期/拘役
- 发现漏罪：先并后减
- 犯新罪：先减后并

#### 缓刑 / 减刑 / 假释
- **缓刑**：拘役、三年以下有期徒刑；累犯和首要分子不适用
- **假释**：有期或无期有悔改，不少于 1/2；考验期 = 剩余刑期（无期为 10 年）

### 9. 重点罪名

| 罪名 | 要点 |
|------|------|
| **抢劫罪** | 暴力盗窃、诈骗、抢夺也按抢劫 |
| **绑架罪** | 要挟对象为第三人 |
| **盗窃罪** | 他人财物和平占有 |
| **侵占罪** | 代为保管、他人遗忘或埋藏物非法占有 |
| **抢夺罪** | 对物实施暴力（携带凶器 → 抢劫罪） |

## 六、其他法律责任

### 消费者权益
- 商品/服务欺诈 → **3 倍**赔偿；增加赔偿不足 500 元 → 500 元
- 明知缺陷造成死亡或健康损害 → 有权要求赔偿 + 惩罚性赔偿

### 不正当竞争
- 不正当有奖销售：信息不明、谎称有奖、有内定、最高金额 > 5 万元
- 网络不正当竞争（强制跳链接、安装软件等）→ 可判 300 万以下赔偿

## 七、劳动合同法

### 合同类型
- 固定期限 / 以工作任务为期限 / **无固定期限**
- **无固定期限条件**：协商一致、满 10 年、连续 2 次固定期限合同
- ⚠️ 满 1 年不订立书面劳动合同 → **视为无固定期限**

### 工作时间与加班

| 情况 | 报酬 |
|------|------|
| 延长工作时间 | **1.5 倍** |
| 休息日 | **2 倍** |
| 法定节假日 | **3 倍** |

- 每日 ≤ 8 小时，平均每周 ≤ 44 小时

### 非全日制用工
- 每日 < 4 小时，每周累计 ≤ 24 小时
- 可口头协议、无试用期、随时终止不用补偿、支付周期 ≤ 15 天

### 解除与赔偿
- **非过错性解除**：提前 30 日书面通知或额外支付 1 个月工资
- **未签合同**：超 1 月不满 1 年 → 每月支付 **2 倍工资**
- **违法解除**：支付经济补偿标准的 **2 倍**赔偿

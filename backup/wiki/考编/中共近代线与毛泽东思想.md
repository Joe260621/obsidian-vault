## 一、时间线总览

### 建党初期 (1921-1927)

| 时间 | 事件 | 要点 |
|------|------|------|
| **1921.7** | **中共一大**（上海） | 确定党名；12 代表；50+ 党员 |
| **1922** | 中共二大 | 确定党纲 |
| **1923** | **中共三大**（广州） | 确定国共合作纲领；政治基础：新三民主义 |
| **1925.1** | 中共四大（上海） | 无产阶级领导权 + 工农联盟 |

| **1927.3** | 毛泽东《湖南农民运动考察报告》 | 党在农村的阶级路线（**毛思想萌芽**） |
| **1927.4.27** | 中共五大（汉口） | 土地革命时期开始 (1927-1937) |

**路线**：渡湘江 → 强渡乌江 → 占遵义 → 四渡赤水 → 巧渡金沙江 → 抢大渡河 → 飞夺泸定桥 → 夹金山 → 过草地 → 会宁会师 / 吴起镇会师

**遵义会议**（毛思想成熟标志）

## 二、毛泽东思想

### 发展的四个阶段

| 阶段 | 时间 | 代表著作 |
|------|------|----------|
| **萌芽** | 1925-1927 | 《中国社会各阶级的分析》《湖南农民运动考察报告》 |
| **形成** | 1927-1935 | 《红色政权为什么存在》《井冈山斗争》《星星之火，可以燎原》《反对本本主义》 |
| **成熟** | 1935-1945 | 《实践论》《矛盾论》《论持久战》《新民主主义论》 |
| **确立** | 1945 中共七大 | 写入党章 |

### 活的灵魂

| 核心 | 地位 |
|------|------|
| 🎯 **实事求是** | **精髓** |
| 🎯 **群众路线** | **核心内容**：一切为了群众，一切依靠群众 |
| 🎯 **独立自主** | **立足点** |

### 三大法宝（《共产党人发刊词》）

1. 🎯 **武装斗争**
2. 🎯 **统一战线**（1929 年周恩来《给红四军前卫的指示信》首次提出）
3. 🎯 **党的建设**（组织建设是物质基础）

## 备考要点速记

1. 🔑 南昌起义 = 武装反抗第一枪；八七会议 = 枪杆子里出政权
2. 🔑 秋收起义 → 首次公开打出中共旗帜；三湾改编 → 党指挥枪
3. 🔑 第一个根据地：井冈山；首部毛选：《反对本本主义》
4. 🔑 遵义会议 → 成熟转折；七大 → 写入党章
5. 🔑 活的灵魂：实事求是（精髓）、群众路线（核心）、独立自主（立足点）
6. 🔑 三大法宝：武装斗争、统一战线、党的建设
7. 🔑 七届二中全会：两转折 + 两务必 + 执政党建设
8. 🔑 毛泽东思想首次提出：1943 王稼祥；首次系统归纳：1945 刘少奇

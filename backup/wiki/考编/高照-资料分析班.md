## 核心公式速查

### 基期与现期

| 需求 | 公式 | 条件 |
|------|------|------|
| 基期量 | `基期 = 现期/(1+r)` | — |
| 化除为乘 | `基期 ≈ 现期×(1-r)` | \|r\|<5% |
| 现期量 | `现期 = 基期×(1+r)` | — |
| 保持增长率 | 利滚利：逐年乘 (1+r) | r>0 |

### 增长率

| 类型 | 公式 |
|------|------|
| 间隔增长率 | `r = r1 + r2 + r1×r2` |
| 混合增长率 | 居中，偏向基期量大的一方 |
| 乘积增长率 | `rA = rB + rC + rB×rC` |
| 名义/实际 | `(1+r名义) = (1+r实际)×(1+CPI)` |

### 增长量

| 方法 | 说明 |
|------|------|
| 公式 | `增长量 = 现期/(1+r) × r` |
| 百化分 | r → 1/n，增长量 = 现期/(n+1) |
| 比较 | 现大率大则增量大（大大则大） |

### 比重

| 类型 | 要点 |
|------|------|
| 基期比重 | `A/B × (1+b)/(1+a)` |
| 两期比重升降 | a > b → 比重上升 |
| 升降百分点 | 选 < \|a-b\| 的选项 |

### 年均增长

- **年均增长率**：r<10% 用估算（间隔思维代入）；r>10% 用平方数居中代入
- **年份差选择**：2011-2015 年份差为 4，基期为 2011 年
- **常用幂次**：1.2³≈1.7、1.2⁴≈2.0、1.2⁵≈2.5

## 高频题型

### 1. 化除为乘（基期快速计算）
> 应用环境：求基期、差距小、|r|<5%。技巧：先变正负，再拆括号。

### 2. 混合增长率
> 进口 + 出口 = 进出口；人民币 + 外币 = 本外币
> ⚠️ 警惕反例：99.9% 情况用现期代替基期判断偏向

### 3. 两期比重
> 判定：两个时间 + 比重 + 升降
> 公式：现期比重 - 基期比重 ≈ A/B × (a-b)/(1+a)
> 秒杀：a > b 上升，a < b 下降

### 4. A和非A思想
> 变型1：A和非A在比重中的运用
> 变型2：A和非A在乘积增长率中运用（求部分增长率）

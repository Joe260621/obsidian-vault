# 马克思主义哲学

> 来源：幕布文档剪藏 | 整理日期：2026-06-20

---

## 一、哲学基本问题

**哲学**：理论化、系统化的世界观（自然、社会和思维知识的概括和总结），研究世界发展最一般规律，**以具体科学为基础**。

### 基本问题：思维和存在（精神和物质）的关系

| 维度 | 划分 | 说明 |
|------|------|------|
| 谁为第一性 | **唯物主义** vs **唯心主义** | 恩格斯提出 |
| 是否存在同一性 | **可知论** vs **不可知论** | 唯物主义和彻底的唯心主义都是可知论 |

## 三、唯物主义

### 1. 古代朴素唯物主义
- **特征**：把 1 种或多种具体物质（水、火、气等）视为世界本原
- **局限**：非科学性、直观性、猜测性（但有辩证法思想）

| 代表人物 | 名言 |
|----------|------|
| 泰勒斯 | 水是万物的始基 |
| 荀子 | 天地合而万物生，阴阳接而变幻起 |
| 王夫之 | 气者，理之依也 |

### 2. 形而上学唯物主义
- **特征**：把基本粒子的构成、物质结构或某种属性作为世界本原
- **局限**：机械性、不懂辩证法、不彻底性
- **关键词**：孤立、静止、片面

### 3. 辩证唯物主义和历史唯物主义（马克思主义）

- **最根本的世界观和方法论**：实践唯物主义
- **活的灵魂**：具体问题具体分析

#### 三大理论来源

| 来源 | 内容 |
|------|------|
| 德国古典哲学 | 黑格尔辩证法（合理内核）+ 费尔巴哈唯物主义（基本内核） |
| 英国古典政治经济学 | — |
| 英法空想社会主义 | 法：圣西门、傅立叶；英：欧文 |

#### 两大理论贡献
- **剩余价值学说**
- **唯物史观**

## 四、物质与意识

### 物质运动与时空

| 概念 | 定义 |
|------|------|
| **时间** | 物质运动过程的持续性、顺序性（**一维性**） |
| **空间** | 运动着的物质的伸张性、广延性（**三维性**） |
| **运动** | 绝对的（无时无刻都在运动） |

### 意识

- **本质**：客观存在的主观反映
- **起源**：自然界长期发展 + 社会历史的产物
- 🎯 **劳动**在意识产生中起**决定性作用**
- **物质决定意识**，物质与精神相对独立
- **意识的能力性**：目的性、计划性、创造性、对人体活动控制、对客观世界改造

## 五、辩证法

### 两大特征：联系 + 发展

**联系**：事物内部之间相互影响、作用和制约
- 客观性、普遍性、多样性、条件性

### 三大规律

#### 1. 对立统一规律 —— 源泉和动力（实质和核心）

| 概念 | 要点 |
|------|------|
| **同一性** | 相对、有条件；矛盾双方相互联系、相互吸引 |
| **斗争性** | 绝对、无条件；矛盾双方相互排斥、相互对立 |
| **内因** | 事物变化的根据和**根本原因** |
| **外因** | 事物之间的联系和影响，必须通过内因起作用 |

**矛盾的普遍性与特殊性**：
- 普遍性（共性）：无条件、绝对 → **两点论**
- 特殊性（个性）：有条件、相对 → **具体问题具体分析**
- 📌 普遍性**寓于**特殊性之中
- 📌 承认矛盾普遍性是科学认识的**首要前提**
- 📌 **矛盾分析法**是唯物辩证法的**根本方法**

**矛盾发展的不平衡性**：

| 概念 | 关键词（题眼） |
|------|----------------|
| **主要矛盾** | 重点、中心、核心、基础、首位、突破口 |
| **主要方面** | 本质、主流、方向、大局、实质、优势、总的来说 |

#### 2. 质量互变规律 —— 发展的基本形式和状态
- 量变与质变互相渗透

#### 3. 否定之否定规律 —— 发展方向和道路
- 自我否定、扬长弃短
- **螺旋式上升**或**波浪式前进**

### 五对范畴

| 范畴 | 要点 |
|------|------|
| 现象和本质 | — |
| 内容和形式 | — |
| 原因和结果 | — |
| 偶然性和必然性 | 必然性**寓于**偶然性；一定条件可转化；杰出人物出现具必然性，其影响是偶然 |
| 可能性和现实性 | 揭示客观事物由可能向现实转化过程 |

## 六、认识论

### 实践

- **实践决定认识** — 实践具有**直接现实性、自觉的能动性、社会历史性**
- 🎯 **劳动**对认识的发生起**决定性**作用

#### 三种实践形式
1. **生产劳动实践**（物质生产实践）—— 最基本
2. 处理社会关系的实践（社会政治实践）
3. 科学实验（科学文化实践）

- 📌 从实际出发是发挥主观能动性的**前提**
- 📌 实践是发挥主观能动性的**基本途径**

### 真理

| 属性 | 说明 |
|------|------|
| **客观性** | 真理内容不依赖主体 |
| **绝对性** | 真理确实存在 |
| **相对性** | 相对于特定事物和历史条件 |

- 真理与谬论：相互对立、相互转化
- **实践是检验真理的唯一标准**
- 实践标准的确定性与不确定性：试了就知道 vs 不一定能试（历史条件限制）

## 七、历史唯物论

### 社会存在与社会意识

- **社会存在决定社会意识**
- 社会意识具有**相对独立性**：发展不平等性、历史继承性、能动反作用

### 社会意识的构成

| 层次 | 内容 |
|------|------|
| 社会心理 | 感知、情绪、心态、习俗 |
| 社会意识形态 | 法律、道德、宗教、艺术 |

- **意识形态 vs 非意识形态**：是否反映经济关系、阶级关系

### 生产力与生产关系

| 概念 | 内容 |
|------|------|
| **生产力** | 人与自然的关系；要素：劳动对象、劳动资料、**劳动者（决定性力量）** |
| **生产关系** | 人与人的关系；生产资料所有制关系、产品分配关系 |

- 📌 **生产力决定生产关系**
- 🎯 **生产方式**是社会历史发展的**决定力量**
- 🎯 **劳动**是理解全部发展史的钥匙 —— 恩格斯

### 经济基础与上层建筑

- **经济基础决定上层建筑**
- 经济基础：占统治地位的生产关系的总和
- 上层建筑：社会意识形态 + 政治法律制度和设施
- 📌 **国家政权**是上层建筑的**核心**

### 社会形态更替

- 三大阶段：人依赖社会 → 物依赖社会 → 个人全面发展的社会
- 特征：必然性 + 历史选择性

### 社会历史发展动力

| 动力类型 | 地位 |
|----------|------|
| 社会基本矛盾 | **根本动力** |
| 阶级斗争 | **直接动力** |
| 社会革命 | 国家政权是根本问题 |
| 改革 | — |
| **科学技术革命** | 推动经济和社会发展的**强大杠杆** |

### 阶级

- 剩余产品的出现 → 阶级产生的**物质前提**
- **生产资料私有制的确定** → 阶级产生的**直接原因**
- 人们对**生产资料占有关系不同** → 阶级划分**最具决定意义的标准**

## 备考要点速记

1. 🔑 哲学基本问题：思维和存在 → 第一性（唯物/唯心）+ 同一性（可知/不可知）
2. 🔑 主观唯心：人的精神 → 王阳明、笛卡尔；客观唯心：客观精神 → 老子、黑格尔、柏拉图
3. 🔑 马克思主义：实践唯物主义（世界观）+ 具体问题具体分析（灵魂）
4. 🔑 三大来源：德国古典哲学 + 英国古典政治经济学 + 英法空想社会主义
5. 🔑 两大贡献：剩余价值学说 + 唯物史观
6. 🔑 劳动 → 意识产生 + 认识发生的决定性作用
7. 🔑 对立统一规律 = 实质和核心；矛盾分析法 = 根本方法
8. 🔑 主次矛盾（重点/中心/核心）vs 主次方面（本质/主流/大局）
9. 🔑 生产力决定生产关系；经济基础决定上层建筑
10. 🔑 国家本质：统治阶级的工具；核心：国家政权

## 一句话总结

**亚朵用"12个客户旅程触点 + 一线授权 + 平台化会员生态"，把酒店从住宿工具做成了生活方式品牌——5400万会员、62.5%自有渠道占比，是酒店行业全域运营的标杆。**

## 与希尔顿模式的对比

| 维度 | 希尔顿（传统五星） | 亚朵（新零售酒店） |
|------|-----------------|-----------------|
| **会员规模** | 品牌会员体系 | A-Card 5400万注册会员 |
| **自有渠道占比** | 依赖OTA | CRS 62.5%（行业罕见） |
| **运营重心** | 大促驱动（春促/秋促） | 触点SOP驱动（12个关键时刻） |
| **一线授权** | 标准化流程 | 300-500元自主决策权 |
| **品牌定位** | 高端商务 | 标签化：阅读+摄影 |
| **会员生态** | 酒店场景内闭环 | 酒店场景外延伸：出行/阅读/运动/饮食/艺术 |
| **营收延伸** | 客房+餐饮+康体 | 床品扫码购买+联名IP+会员电商 |

## 三大核心方法论

### 1. 客户旅程SOP——12个关键时刻

亚朵把客户体验拆解为12个触点，每个触点有标准动作：

```
预订 → 走进大堂（奉茶/3分钟入住/免费升舱）→ 进房间第一眼
→ 服务咨询 → 早餐 → 等人等车 → 加餐（深夜粥到21:00-23:00）
→ 离店（送水/冬天温水）→ 点评 → 第二次想起 → 推荐朋友 → 再预订
```

**可借鉴点：** 希尔顿微商城运营中对应的是"会员日+积分到期提醒+生日权益"的触点体系。区别在于亚朵把触点从线上延伸到了线下空间体验。

### 2. 一线授权——300-500元自主决策

亚朵给一线服务人员300-500元的自主决策权限，用于解决客户的小问题（忘记行李、临时需求等），无需层层审批。

**可借鉴点：** 这是会员运营中"差异化服务"在线下的落地——高等级会员的即时服务响应。对应希尔顿项目中"高净值VIP一对一服务"的思路，但亚朵给了具体金额和决策机制。

### 3. 平台化会员生态——从"酒店会员"到"生活方式会员"

```
传统酒店会员 = 房价折扣 + 积分 + 延迟退房
亚朵A-Card = 上述 + 免费借书 + 外部品牌权益 + 主题活动 + 床品购买
```

- 与KEEP、永璞咖啡、单向空间、高德打车等品牌打通权益
- 各大平台权益互通，简化入会流程
- 酒店场景内的产品（床品）延伸为可购买商品
- 2015年OTA占71.8%→亚朵自有渠道CRS达62.5%（反向）

**可借鉴点：** 这是希尔顿项目"客房上架微商城"思路的升级版——不只是把酒店产品放到线上卖，而是把酒店体验变成可购买的品牌商品。

## 面试可用的对比视角

如果有面试官问"你觉得酒店行业运营怎么做"，可以这样答：

> 我在希尔顿做私域运营的时候，主要在做一件事：把酒店商城从大促依赖升级为常态化运营。后来我研究过亚朵的模式，发现他们在另一个维度做了更有意思的事——把酒店会员体系从酒店场景内延伸到了场景外，和KEEP、永璞咖啡这些生活方式品牌打通权益。本质上，希尔顿系在补数字化的课，亚朵在做用户生态的事。这两个方向都是我感兴趣的。

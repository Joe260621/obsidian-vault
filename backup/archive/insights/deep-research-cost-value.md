# deep-research 对抗性验证的价值与成本

> 最后更新: 2026-06-21

## 2026-06-20
- 发现: deep-research 的对抗性验证（adversarial verification）是最贵的环节，也是最核心的价值——没有验证的报告只是整理过的搜索结果。中文轮24源→83声明→4通过验证，消耗239万token（~$1.44）；英文轮API余额不足导致验证全部失败，97万token浪费。
- 教训: 启动 deep-research 前应先检查 API 余额是否够跑完验证。¥45只够跑半轮深度验证。

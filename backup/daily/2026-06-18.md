|---|------|------|------|

## 经验备忘

1. **集思录 QDII API 关键**：不加 `X-Requested-With: XMLHttpRequest` 只返回20条，加上返回全量55条
2. **Playwright 是 SPA 克星**：任何 JS 渲染的页面都能抓，只要登录态持久化
3. **Boss直聘的反爬是降维打击**：连 F12 都禁，别硬碰
4. **CronCreate 的 durable 只能保任务不丢，不能保准时执行**：关机就全没了，必须开机+Claude Code运行
5. **飞书推送链接格式**：手机端点链接直接跳夸克打开，比抓数据更实用

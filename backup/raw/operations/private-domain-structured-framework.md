# 私域运营结构化框架

- **Source**: 幕布笔记 → Inbox/私域运营结构化框架.html
- **Collected**: 2026-06-20

## 一、梳理商品商业通路
业务场景分析：直播平台、小程序商城、公众号、微信社群、企微/个微等

## 二、梳理用户价值养成路径
用户分层、分群运营模型搭建和数据导入
寻找和创造用户需求场景突出产品

## 三、搭建用户成长阶段
认知关系 → 交易关系 → 参与关系 → 共享关系

## 四、梳理用户不同层级最优成长路径
用户画像颗粒度：基本数据/行为数据/营销数据/交易数据

## 五、商品的交易链路分析
曝光 → 点击 → 加购 → 购买 → 复购

## 六、完善最优成长路径的引导和激励策略
最大业务场景：强引导+最大让利营销模型
其他业务场景：轻运营动作设计

## 七、系统构建会员成长体系
- 赋予消费者消费能力积分，兑换相应价值商品或奖励
- 赋予会员身份等级，拥有独家专属特权

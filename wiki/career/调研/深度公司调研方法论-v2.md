# 📋 深度公司调研方法论 — v2.0（创新版）

> **用途**：匹配公司→优先投递→面试准备的标准化深度调研流程
> **核心理念**：不是"搜资料整理"，而是"用投资人视角看公司+用操盘手视角看岗位+用创新者视角提方案"
> **迭代记录**：v1.0 基础调研 → v2.0 加入AI赋能创新体系+KOC运营飞轮+行业通病突围

---

## 一、调研的五个层次

调研深度分5级，**每级都是下一级的基础**：

```
Level 1: 搜资料 (知道有什么)
   ↓
Level 2: 理结构 (知道是什么)
   ↓
Level 3: 挖数据 (知道多少)
   ↓
Level 4: 看透局 (知道为什么)
   ↓
Level 5: 提方案 (知道怎么做+创新)
```

> 大多数人停在Level 1-2（搜公司简介+JD），**面试能赢的人至少到Level 4，创新方案到Level 5。**

---

## 二、调研框架（5大模块）

### 模块A：行业透视

| 问题 | 目的 | 信息来源 |
|:----|:-----|:---------|
| 这个赛道有多大？ | 面试中展示"我懂你行业" | 行业报告、券商研报 |
| 增速多少？天花板在哪？ | 判断公司增长空间 | 艾瑞/亿欧/36氪 |
| 核心玩家是谁？格局如何？ | 对标分析 | 行业报告 |
| 行业核心痛点是什么？ | 引出你的创新方案 | 行业分析+用户评论 |
| 政策/技术/消费趋势是什么？ | 展示前瞻性 | 新闻+政策文件 |

**深度要求**：至少引用**3个以上具体数据**（市场规模、增速、渗透率等）。

### 模块B：公司基本面

| 问题 | 目的 |
|:----|:-----|
| 公司靠什么赚钱？ | 理解商业模式 |
| 老板是谁？什么背景？ | 判断公司文化 |
| 核心竞争优势是什么？ | 面试中展示"我懂你们" |
| 公司处在什么发展阶段？ | 判断岗位空间 |
| 最近有什么大新闻/动作？ | 面试中展示关注度 |

### 模块C：客户/市场格局（新增·关键）

> 🚀 **这是你们面试中拉开差距的核心模块**——大多数候选人只调研公司本身，不调研公司的客户。

| 维度 | 调研内容 | 为什么重要 |
|:-----|:---------|:-----------|
| **甲方是谁？** | 公司服务的核心客户（尤其是乙方公司） | 面试时说"我知道贵司的核心客户是谁、什么状态" |
| **客户的市场表现** | 销量、份额、增长/下滑趋势 | 展示你懂他们的生意 |
| **客户的客户** | 最终用户画像、需求变化 | 展示你懂用户运营的终局 |
| **客户面临的挑战** | 市场压力、竞争、转型困境 | 为你的创新方案埋下伏笔 |
| **举例**：如果调研智合创享，要调研广汽丰田/本田的市场现状 | | |

### 模块D：岗位与匹配度（6维评估）

从6个维度评估你的匹配度，并在面试中主动引导：

| 维度 | 权重 | 自评方法 |
|:-----|:----:|:---------|
| 硬技能匹配 | ⭐⭐⭐⭐⭐ | JD逐条对应，有案例支撑 |
| 行业经验 | ⭐⭐⭐⭐ | 有=优势，无=准备跨行业话术 |
| 管理经验 | ⭐⭐⭐ | 团队规模、管理方法 |
| 行业认知 | ⭐⭐⭐⭐ | 面试中引用具体数据 |
| 创新能力 | ⭐⭐⭐⭐⭐ | **面试突围关键**——你的创新方案 |
| 文化契合 | ⭐⭐⭐ | 通过背调/口碑判断 |

### 模块E：创新方案 — 面试突围核武器 🔥

> **这是v2.0升级的核心**。当HR问"你有什么想法"时，其他候选人说"我会做好用户分层和活动运营"，你说的是下面这套系统。

#### E1 先指出行业通病（建立认知差）

```
"我调研后发现，贵司这个行业/KOC运营普遍有三个问题..."
```

找到2-3个行业痛点，这句话能让你立刻从"我是来应聘的"变成"我是来解决问题的"。

#### E2 再给创新框架（展示体系化思维）

```
"所以我提出了一个XX模型/体系/飞轮..."
```

不要只讲一个点，要讲一个体系。推荐格式：
- **一句话命名**：让人记住（如"KOC-OS飞轮""数据热力地图""双引擎驱动"）
- **一张图**：面试中能画出来（流程图/飞轮/矩阵）
- **一个案例**：你之前在哪验证过

#### E3 最后落到该公司（展示落地能力）

```
"这个方案在贵司可以分三步走..."
```

让HR觉得："这人不是来吹概念的，是来干活的。"

#### E4 创新方案必须包含的加分要素

| 要素 | 为什么加分 | 你的独特优势 |
|:-----|:-----------|:-------------|
| **AI赋能** | 2026年AI是运营必答题，但90%候选人不会谈 | ✅ 你系统性验证过AI提效 |
| **数据驱动** | 不是"我看数据"，而是"数据怎么反哺决策" | ✅ 你的RFM+SCRM实战经验 |
| **跨行业视角** | 单一行业的人容易被惯性束缚 | ✅ 你跨了4个行业 |
| **可量化指标** | 要能说清楚"这个方案能带来什么效果" | ✅ 你的数据意识 |

## 三、调研执行流程（SOP）

### Step 1：搜
| 渠道 | 搜什么 | 产出 |
|:-----|:-------|:-----|
| 百度/知乎 | 公司名+背调/口碑/评价 | 员工口碑、避坑信息 |
| 36氪/亿欧 | 公司名+行业赛道 | 行业报告、融资新闻 |
| 猎聘/BOSS | 岗位JD+公司主页 | 岗位要求、薪资范围 |
| 金鼠标/广告门 | 公司名+案例（乙方公司） | 真实项目案例 |
| 天眼查/企查查 | 公司基本信息 | 股权结构、风险信息 |
| 微信搜一搜 | 公司名+行业趋势 | 行业深度文章 |

### Step 2：理
整理为结构化文档，按以下顺序：

```
公司信息卡（一页速览）
├── 行业背景（赛道规模+趋势+数据）
├── 公司基本面（商业模式+团队+优势）
├── 客户/市场格局（如乙方公司 + 甲方市场分析）
├── KOC/创新运营能力（已有案例+可改进点）
├── 岗位JD分析（要求+匹配度+可能追问）
├── 面试策略（核心卖点+话术+准备清单）
└── 🚀 创新方案（行业通病+创新框架+落地路径）
```

### Step 3：备
- 准备**3个必问问题**的回答（跨行业经验、为什么选我们、你的创新想法）
- 准备**1个你给该公司的创新方案**（2分钟说完）
- 准备**1个行业数据引用**（面试中脱口而出）

### Step 4：模拟
- 对着镜子/录音说一遍面试核心话术
- 检查："如果我是HR，我会被这段话打动吗？"

## 四、面试"三句话法"——最后的临场武器

> 任何面试中——当HR问"你对我们公司/这个岗位有什么想法？"——用这个结构回答：

| 句 | 内容 | 时长 | 效果 |
|:--:|:-----|:----:|:-----|
| **第一句** | 指出行业通病/你洞察到的问题 | 20s | ✅ 建立"你有深度思考"的印象 |
| **第二句** | 你的创新框架/方法论（命名+逻辑） | 40s | ✅ 展示"你有体系化能力" |
| **第三句** | 落到该公司怎么落地的具体建议 | 30s | ✅ 展示"你能干活" |

**总时长控制在90秒以内**，然后问HR"您觉得这个方向怎么样？"——把对话变成双向交流。

## 五、创新方案素材库（持续更新）

> 以下是可以组合使用的"创新组件"，根据目标公司行业灵活搭配。

### 组件1：数据热力地图
- **适用场景**：APP社区运营、内容运营
- **核心逻辑**：用户行为数据→内容选题指令→精准内容生产
- **举例**："用社区搜索热点告诉KOC该发什么，而不是让KOC自己猜"
- **来源**：智合创享调研

### 组件2：游戏化成长体系
- **适用场景**：KOC运营、用户运营
- **核心逻辑**：青铜→传说等级+技能树+解锁权益
- **举例**："让KOC像打游戏一样升级，每个等级解锁新工具和新权益"
- **来源**：智合创享调研+蔚来案例

### 组件3：AI人机协同赋能
- **适用场景**：任何运营场景（最强差异化武器）
- **核心逻辑**：AI工作坊（选题/脚本/剪辑/数据全链路）
- **举例**："2026年我系统性验证了AI在内容生产提效3-5倍"
- **来源**：你的AI实践经验

### 组件4：KOC-OE飞轮
- **适用场景**：KOC运营体系搭建
- **核心逻辑**：社区数据→内容生产→多平台分发→线索转化→数据回流→社区数据
- **举例**：智合创享I-TOP平台升级应用
- **来源**：智合创享调研

### 组件5："反KOC"信任机制
- **适用场景**：品牌口碑运营
- **核心逻辑**：允许KOC说缺点→建立信任→推荐更有说服力
- **举例**：超哥超车"买个der"效应
- **来源**：超哥超车案例

### 组件6：内容矩阵工厂
- **适用场景**：新媒体内容运营
- **核心逻辑**：标准化模板库+平台差异化分发+KOC个性化填充
- **举例**："像乐高积木一样，KOC只需要填入自己的真实体验"
- **来源**：谦腾经验+行业调研

## 七、重要提醒

### 信息来源可靠性分级

```
⭐⭐⭐⭐⭐ 行业报告（艾瑞/亿欧/36氪）→ 最可靠
⭐⭐⭐⭐  公司官方信息（官网/招聘JD）→ 次可靠
⭐⭐⭐   职场平台（脉脉/知乎/牛客）→ 偏主观，交叉验证
⭐⭐     自媒体/论坛 → 仅供参考
```

### 背调风险评估

面试前必须确认以下风险：
- [ ] 公司是否有劳动纠纷/负面新闻？
- [ ] 是否强制大小周/高强度加班？
- [ ] 员工口碑如何？（脉脉/知乎/看准）
- [ ] 是否有合规风险？（医美/大健康/金融行业尤其注意）
- [ ] 是否有竞业限制风险？
- [ ] 背调严格度（第三方机构/HR内部核实/hr问前公司）

风险评估结果决定该公司的投递优先级：
- ✅ **风险低**：正常投递
- ⚠️ **有风险但可控**：练手公司/保底选择
- ❌ **风险高**：存公司库，暂不投递

## 一、行业分析四步法

### Step 1：用户与利润分配逻辑

| 维度 | 分析 |
|:----|:------|
| **谁付钱？** | 18-35岁年轻白领+家庭客群——追求品质、性价比、地道客家味。客语人均消费约75-79元 |
| **消费特点** | 高频低客单（餐饮天然属性），工作日午/晚餐+周末家庭聚餐，决策快、就近消费 |
| **钱怎么分？** | 食材成本约35-40%，人工20-25%（30-40人/店），租金15-20%（购物中心），净利润率8-15% |
| **利润集中在哪？** | **效率驱动**——翻台率（正餐日均1.5-2.5轮）+客单价（75-79元）×门店数（90家）=营收模型 |
| **餐饮私域价值** | 私域会员LTV是公域新客的**3.2倍**，获客成本仅1/10。头部品牌私域贡献率超30% |

### Step 2：市场与行业宏观分析

| 维度 | 数据 |
|:----|:------|
| **餐饮市场规模** | 2025年中国餐饮市场约**5.8万亿元**（+3.2%），2026年预计突破**6万亿元** |
| **餐饮连锁化率** | 25%（2019年仅13%→翻倍），连锁化趋势加速 |
| **正餐/中餐连锁** | 中餐正餐占比约30%，客家菜是细分品类中的区域性特色 |
| **闭店潮** | 2025年超**300万家**门店退场，闭店率35%——行业出清加速 |
| **行业阶段** | 成熟期但结构性机会仍在——连锁化率提升+数字化升级是主旋律 |
| **私域渗透率** | 头部品牌私域贡献30%+，但**24%品牌私域贡献不足5%**——分化严重 |

### Step 3：竞争格局与头部企业

**客家菜赛道竞争格局：**

| 品牌 | 门店 | 定位 | 优势 |
|:----|:----:|:-----|:-----|
| **客语** | ~90家 | 时尚品质客家菜 | 规模最大+购物中心渠道+品牌力 |
| 恒信客家王（前身） | — | 传统客家菜 | 广州知名度，但已品牌升级为客语 |
| 客家班 | ~20家 | 新派客家菜 | 年轻化但规模小 |
| 各地方单店 | — | 传统客家菜 | 性价比高但无品牌力 |

**餐饮连锁整体格局：**
- 中餐正餐头部：西贝（300+家）、太二（200+家）、客语（~90家）——品类不同但同属正餐连锁赛道
- 客家菜是个相对稀缺的品类——没有全国性大品牌，客语在规模和认知上是唯一的头部

### Step 4：关键变量分析

| 变量 | 影响 | 对客语的含义 |
|:----|:------|:------------|

### 模块C：客户/市场格局

| 维度 | 分析 |
|:-----|:------|

### 模块E：创新方案（面试核武器）

#### E1 行业通病（3个洞察）

```
餐饮连锁做私域普遍存在三个问题（我开过店我太懂了）：

痛点1：门店配合度差
  → 店长KPI是翻台率和营业额，私域拉新=额外负担
  → 让门店推会员/加企微，必须'简单到看一眼能做+有激励'

痛点2：内容同质化严重
  → 餐饮私域=发优惠券+推新菜，用户很快疲劳
  → 需要'有温度的内容'——后厨故事、食材溯源、客家文化

痛点3：线上线下割裂
  → 公域（美团/抖音）引流到店→吃完就走了
  → 用户离开门店后品牌失联，复购靠'下次路过'
  → 私域要解决的问题是：离店后怎么持续触达
```

#### E2 创新框架：「客语·烟火气私域飞轮」

```
核心逻辑：餐饮私域不是"线上卖券"，而是"把门店的温度延伸到离店后"

               ┌──────────────────────┐
               │  门店引流·触点铺设    │ ← 桌贴+服务员话术+扫码点餐
               └──────────┬───────────┘
                          ↓
               ┌──────────────────────┐
               │  企微社群·有温度内容   │ ← 食材故事+客家文化+后厨日常
               └──────────┬───────────┘
                          ↓
          ┌─────────────────────────────────┐
          │  会员体系·等级权益（关键创新）    │
          │  L1尝鲜→L2常客→L3老友→L4挚友  │
          │  每一级解锁一道隐藏菜/专属折扣   │
          └──────────────┬─────────────────┘
                          ↓
               ┌──────────────────────┐
               │  复购驱动·场景触达      │
               │  生日餐/纪念日/节气菜   │
               └──────────┬───────────┘
                          ↓
               ┌──────────────────────┐
               │  高价值会员·KOC裂变     │
               │  带朋友来=积分翻倍      │
               └──────────────────────┘
```

**面试话术（40秒版）：**
> "我开过2家餐饮店，太懂餐饮做私域的痛点了。核心不是搞多复杂的系统，而是三件事：一、让门店愿意推——拉新激励直接算门店业绩，话术设计成'1秒能说完'；二、让用户愿意留——群内容不全是优惠券，而是客家菜的故事、食材的溯源、后厨的日常；三、让会员有升级动力——每一级都解锁一个'非会员吃不到的隐藏菜'。这三点通了，私域就转起来了。"

#### E3 落地三步走

| 步 | 周期 | 动作 | KPI |
|:--:|:----:|:------|:----:|
| **第1步** | 1个月内 | 设计门店引流SOP（桌贴+话术+激励）+搭建企微社群框架 | 单店日均加粉20+，90店覆盖 |
| **第2步** | 2-3个月 | 上线会员等级体系+L1-L4权益+场景化触达（生日/纪念日/节气） | 会员复购率+15%，社群活跃率30%+ |
| **第3步** | 4-6个月 | AI内容库搭建（客家文化×新菜推广×节日营销）+KOC裂变机制 | 私域贡献堂食GMV 10%+，KOC转介绍率20%+ |

#### E4 加分要素

| 要素 | 你的筹码 |
|:-----|:---------|
| **开过2家餐饮店** | 全时段（早中晚餐）和单品聚焦（单品类）两种模式——懂门店运营的每个环节 |
| **AI赋能** | AI内容批量生成（菜谱/食材故事/客家文化/节日营销）3-5倍提效 |
| **会员体系已验证** | 万富4级VIP体系+硕牛RFM——餐饮会员体系可直接迁移 |
| **跨行业视角** | 服务/零售/电商/酒店——知道餐饮私域能怎么做、不能怎么做 |

## 三、面试应用

### 三句话法（90秒黄金回答）

> 当HR问"你觉得客语的私域应该怎么做？"

| 句 | 内容 | 时长 |
|:--:|:------|:----:|
| **第一句** | "开过2家店之后我特别明白一点——餐饮做私域门店配合是命门。如果让服务员觉得'又要多干活'，什么系统都白搭。所以我的策略是：SOP简单到看一眼就能做，拉新激励直接算门店业绩。" | 30s |
| **第二句** | "打法上分三层：底层是触点铺设——桌贴+服务员话术+扫码点餐，让每个到店客都能进私域；中层是社群内容——客家菜故事+食材溯源+后厨日常，让用户有温度感知；顶层是会员等级体系——L1到L4每级解锁一个'隐藏菜'，让复购有驱动力。" | 35s |
| **第三句** | "第一步落地门店引流SOP+企微社群框架（1个月），第二步上会员体系+场景触达（2-3个月），第三步AI内容库+KOC裂变（4-6个月）。核心目标是6个月私域贡献堂食10%+。" | 25s |

### ⚠️ 可能追问+回答方向

| 追问 | 回答要点 |
|:-----|:---------|
| "你开过什么店？为什么关了？" | "第一家试水全时段（早餐/午餐/晚餐/宵夜全做）——发现精力太分散，什么时段都没做好。第二家改单品聚焦模式——只做午晚两市，精简菜单，效率明显提升。后来因为个人发展转向运营方向，才把店转让了。这段经历让我比纯运营背景的人更懂门店——我知道什么方案门店能执行、什么方案是纸上谈兵。" |
| "餐饮私域和零售私域有什么不同？" | "餐饮是高频低客单、决策快、就近消费——所以私域的核心不是拉新（门店自然客流就够了），而是'复购+提频+客单价'。零售私域靠内容种草，餐饮私域靠'场景触发'——生日来吃、纪念日来吃、今天不想做饭来吃——三个场景做透就够了。" |
| "客语目前私域是空白的，你怎么从0搭？" | "不需要从0烧钱。餐饮做私域最便宜的入口是桌贴+服务员一句话+扫码点餐的会员弹窗——这些都是零成本流量。先在3家标杆店跑通SOP，验证后再全国复制。" |
| "为什么选客语？" | "我看好客家菜这个品类——有地方特色、有文化根基、有差异化。客语是90家店规模的区域头部，这个体量做私域既能看到效果又不会太复杂。而且包食宿说明公司重视员工——这个细节加分。" |
| "期待薪资？" | "上一份合同15K。客语包食宿相当于省了2-3K，综合来看15-18K我都能接受。我对餐饮赛道有信心，更看重能把事情做成。" |

## 五、策略对齐：面试怎么聊

### 核心定位
> **"我不是'懂私域的运营'——我是'开过店又懂私域的运营'。客语找私域负责人，要么懂餐饮不懂互联网，要么懂互联网不懂餐饮——我两边都懂。这就是最大的差异化。"**

### 关键面试策略

⚠️ **优势展示顺序要注意：**

| 面试阶段 | 展示重点 | 为什么 |
|:--------|:---------|:-------|
| **开场30秒** | "我自己开过2家餐饮店" | 建立差异化印象——95%的候选人没有这个 |
| **中间环节** | 私域方法论（会员体系+社群SOP+KOC） | 展示你不是"只有开店经验" |
| **深入环节** | AI赋能+创新方案 | 拉开差距——展示你有前瞻性 |
| **收尾** | 问题反问："客语目前的会员系统到什么程度了？" | 展示你能落地 |

### 关于包食宿（如果HR主动提）

> "包食宿对我来说最大的价值不是省钱，而是让我能和门店团队在一起——新店推行私域SOP的时候，我在店里吃住几天，比在办公室开10次会管用。开过店的人都知道：距离一线越近，方案越靠谱。"

### 如果HR问"餐饮行业很辛苦，你能接受吗？"

> "我开过店我知道餐饮的节奏——周末忙、节假日更忙。但私域运营的好处是：越是门店忙的时候，越是加粉的黄金窗口（排队等位时加企微/社群）。我接受的不是'辛苦'，是'有意义地忙'——能看到方案落地、数据涨起来。"
```

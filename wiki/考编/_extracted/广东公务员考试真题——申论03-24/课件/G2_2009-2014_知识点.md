# 广东申论真题提炼 — Group2: 2009-2014（转型成熟期）

## 时代特征概述

2009-2014年是广东申论考试的转型成熟期，与前一个阶段（2003-2008年探索期）相比呈现以下特点：

1. **选题多元化与社会热点同步**：从2009年的地铁安全事故、2010年的农民工用工短缺、2011年的职业病防治、2012年的市场经济中的政府角色、2013年的社会管理创新，到2014年的绿色发展，每年主题不同，紧跟国家政策导向和社会热点，不再局限于单一的社会问题范畴。

2. **材料体量与复杂度增加**：材料从早期的3-5则增加到8-11则，信息量大幅提升，且材料来源多样化（新闻报道、政府文件、专家观点、案例调查、座谈会记录等），考查考生在复杂信息中提取要点的能力。

3. **题型逐步分化与精细化**：2009-2013年保持3题模式，2014年首次出现4题模式（增加了一道10分的小题）。前两题从单一的概括归纳发展为概括、分析、对比、对策等多种题型并存，大作文仍保持50分的主导地位。

4. **题目要求更具体明确**：从早期的宽泛要求发展为有明确的字数限制、分值标注、作答方向限定（如"仅限材料3和材料4"），评分标准也更细化（2010年第二套答案附带赋分说明）。

5. **参考答案规范化程度提高**：2010年开始出现带分值的赋分说明，2012年以后参考答案结构更清晰，大作文范文质量明显提升，分论点明确、论证逻辑严密。

6. **广东省情紧密结合**：从2010年的广东"双转移"政策、2012年的广东"三打两建"、2014年的广东省主体功能区规划，到韶关绿色转型等，越来越强调对广东省政策的理解和运用。

## 2009年

### 考试概况
- **主题词**：地铁建设、安全生产、地陷事故
- **材料主题**：以H市地铁1号线湘湖站地陷事故为切入点，探讨地铁建设中的安全监管、施工管理、利益博弈等问题
- **材料数**：8则 | **题目数**：3题 | **满分**：100分
- **时限**：120分钟

### 给定材料概要
| 材料 | 核心内容 | 关键信息 |
|------|----------|----------|
| 材料1 | 国家4万亿经济刺激计划、城轨快速发展背景 | 40余座城市规划城轨，3400公里线路，经验不足存隐患 |
| 材料2 | 地陷现象及其成因分析 | 50多个城市出现地陷，"88:10:2"规律——人为因素占88% |
| 材料3 | H市地铁1号线湘湖站事故详情 | 21人死亡，流沙地质，施工未遵循分层分段原则，赶工期 |
| 材料4 | 事故前兆及监管缺失 | 裂缝出现一周未处理，向上级汇报等待审批延误；安全通道缺失，培训缺失，层层转包 |
| 材料5 | 施工方与业主方责任争论 | 王院士vs地铁集团：设计缺陷vs施工问题，明挖法vs暗挖法成本之争 |
| 材料6 | 广州地铁安全措施 | 安全监察部、夜间巡查、地质灾害防控小组、第三方监测 |
| 材料7 | 广州地质难题及地铁发展历程 | 三大地质难题（断裂、花岗岩残积层、石灰岩溶洞），盾构法应用 |
| 材料8 | 国外经验借鉴 | 日本立法保障安全生产、布鲁塞尔压缩成本注重安全 |

### 题目分析

#### 第一题（25分）
- **题型**：归纳概括（教训类）
- **作答要求**：概括H市政府应从材料3和材料4所涉事故中吸取的教训；不超过250字
- **答题要点**：
  1. 地铁施工应严格遵循施工原则（分层分段开挖、先支撑后开挖），不能赶工期
  2. 加强安监工作，发现险情及时上报并采取补救措施；设置安全通道；加大专家监测密度
  3. 加强施工管理，加大对新员工培训力度，严禁层层转包行为
  4. 加快培养地铁轨道技术人才，扩大专业管理与技术人员规模
  5. 地铁路线应尽量取直，首选最安全、最快捷的路线，不能随意修改

#### 第二题（25分）
- **题型**：贯彻执行（简报写作）
- **作答要求**：以事故调查组成员身份，就材料5中施工方和业主方对事故责任认定的争论，给H市政府写一份简报；不超过350字
- **答题要点**：
  - 格式：简报标题（关于H市地铁1号线湘湖段施工工地塌陷事故的简报）
  - 施工方观点：地质复杂、设计缺陷、工期短、成本低
  - 业主方观点：设计没问题、工期安排合理、费用由企业自主报价
  - 专家分析："拍脑袋""图便宜""赶工期""领导意见替代科学决策"
  - 结论：两者协作不够

#### 第三题（50分）
- **题型**：议论文（策论文章）
- **题目**：《地铁建设中如何避免地陷》
- **总论点**：进一步加强技术和管理服务，从严防范地铁地陷事故发生
- **分论点框架**：
  1. 做好前期地质勘查和研究工作，确定方案——加强线路埋法与工法研究，找到适合本地地貌的通用方法
  2. 加强对施工流程的监督，规范操作程序——严格遵守施工原则，科学安排工期，设置安全通道
  3. 严格安全生产监管，完善检测体系和应急机制——设立安全监察部门，建立动态评估检测体系
  4. 强化工程管理，加快人才培养——严禁层层转包，加快培养轨道施工人才
  5. 借鉴国外成功经验——立法建立安全生产责任制，全社会培训机制
- **结尾**：始终把保证市民人身财产安全作为修建地铁的大前提

### 参考答案核心框架
- **第一题踩分点**：施工原则（分层分段）→ 安监补救 → 管理培训 → 人才培养 → 路线取直
- **第二题踩分点**：简报格式分+施工方观点+业主方观点+专家分析+结论
- **第三题（大作文）框架**：
  - 开头：地陷事故频发→人为因素为主→亟需加强防范
  - 五条对策：地质勘查→施工监督→安全监管→工程管理→国外借鉴
  - 结尾：始终把安全作为大前提

### 高频规范表达

- "安全事故存在'88：10：2'规律"
- "拍脑袋、图便宜、赶工期、领导意见替代科学决策"

- "严格……监管，完善……体系和……机制"

## 2010年

### 考试概况
- **主题词**：农民工、用工短缺、权益保障
- **材料主题**：从"民工潮"到"用工荒"的转变，探讨农民工权益保障、技能培训、社会保障等问题
- **材料数**：11则 | **题目数**：3题 | **满分**：100分
- **时限**：120分钟
- **特殊说明**：有两套参考答案（解析一和解析二），需分别提取并比较

### 给定材料概要
| 材料 | 核心内容 | 关键信息 |
|------|----------|----------|
| 材料1 | 农民工流动历史回顾 | 从"民工潮"到"规范流动"到"公平流动"，1997金融危机、2001入世 |
| 材料2 | 2004年珠三角首现用工短缺 | 工资待遇低、权益保障差、企业扩张导致缺工；浙江企业加薪留人 |
| 材料3 | 农民工权益保护政策演变 | 总理讨薪、"两为主"子女教育、取消农业税引返乡潮 |
| 材料4 | 产业升级与技工短缺 | 2006-2007年技工短缺严重；技工教育落后、企业重使用轻培训 |
| 材料5 | 社会保障制度建设 | 2006年国务院意见、新《劳动合同法》2008年实施 |
| 材料6 | 2009年经济复苏后用工短缺再现 | 企业高薪招人、规范管理型企业不愁招人 |
| 材料7 | 经济格局变化与"一体两面" | 企业向中西部转移、机会成本上升；产业结构不合理是根本症结 |
| 材料8 | 新生代农民工特点 | 80后占比75%、就业观念转变、更看重工作环境和发展空间 |
| 材料9 | 社保体系与政府工作报告脉络 | 2004-2010年政府工作报告对农民工问题的层层推进 |
| 材料10 | 广东"双转移"战略 | 产业转移+劳动力转移，免费培训农村劳动力 |
| 材料11 | 2010年中央一号文件与养老保险转移接续 | 覆盖城乡的公共就业服务体系，社保关系跨省转移 |

### 题目分析

#### 第一题（30分）
- **题型**：归纳概括（分阶段概括特征及原因）
- **作答要求**：根据材料1-9，概括各阶段用工特征及其原因；不超过350字
- **答题要点（答案A）**：
  - 第一阶段（2001年左右）：取消限制性政策，由"规范流动"向"公平流动"转变；原因：经济复苏+入世
  - 第二阶段（2004年）：珠三角、闽东南、浙东南劳动密集型企业缺年轻女工和熟练工；原因：工资低、权益差、企业扩张
  - 第三阶段（2006-2007年）：沿海地区技工短缺；原因：技工教育落后、企业重使用轻培训
  - 第四阶段（2009年下半年）：沿海地区用工短缺再现；原因：经济复苏、企业向中西部转移、"三农"投入增加
- **答题要点（答案B，含赋分说明）**：
  - 第一阶段特征：各地工厂企业出现不同程度的用工短缺（2分），农民工初显不足（2分）；原因：扩张性财政政策（2分），加入世贸组织（2分）
  - 第二阶段特征：珠三角、闽东南、浙东南（2分）加工制造业地区（2分）；原因：工资待遇、权益保障、企业扩张（2分），取消农业税（2分）
  - 第三阶段特征：沿海经济发达地区（2分）技工短缺（2分）；原因：产业结构升级（1分），技工供给不足（1分），企业轻培训（1分），缺评价体系（1分），缺流动环境（1分）
  - 第四阶段特征：广东各地企业（2分）用工短缺再次出现（2分）；原因：经济复苏（1分），机会成本加大（1分），产业结构不合理（1分），新生代观念转变（1分）

#### 第二题（20分）
- **题型**：综合分析（特点提炼）
- **作答要求**：分析提炼近10年来我国政府不断加强农民工工作的特点；不超过250字
- **答题要点（答案A）**：
  - 初期：取消流动限制，赋予基本劳动权
  - 2004-2005年：建立工资支付机制，解决欠薪问题
  - 2006年至今：改善就业工作条件、建立社会保障制度、解决住房和子女教育、技能培训、推进各项保险
- **答题要点（答案B，核心得分类键词：及时性、针对性、先进性）**：
  - 及时性：国家及时主动取消流动限制，工资问题出现后及时提出解决方案
  - 针对性：针对住房、子女教育、技能培训、保险等各个问题分别解决
  - 先进性：从解决基本需求→有尊严地生活→有技能有尊严地生活，层层递进
  - 赋分说明：答对及时性、先进性、针对性三个词并阐述可得17分

#### 第三题（50分）
- **题型**：议论文（策论文章）
- **题目**：《进一步加强农民工工作》
- **答案A框架**：
  - 开头：农民工是经济社会发展的重要力量
  - 对策一：加强工资支付管理和监督（监控预警、经济行政法律手段加大处罚）
  - 对策二：统筹解决子女接受义务教育问题（纳入当地教育规划、公办学校为主）
  - 对策三：改革创新社会保障制度（优先解决工伤和医疗、逐步解决养老）
  - 对策四：推进配套制度改革（深化户籍改革、放宽落户条件）
  - 对策五：加强技能培训（纳入规划、组合运用补贴和津贴政策）
  - 结尾：改善条件吸引农民工、加强培训培养优秀队伍
- **答案B框架（含赋分说明）**：
  - 开头：提出问题（农民工短缺现象）+分析原因（户籍限制、企业近利、新生代观念转变、高房价等）
  - 对策一：推进户籍制度改革（4分），取消不合理限制（2分），引导有序流动（2分）
  - 对策二：监督企业提高待遇、保障权益（4分），改善工作环境、提高工资（2+2分）
  - 对策三：完善社保体系（4分），子女上学（1分）、医疗工伤失业养老（1+1+1+1分）、居住条件（1分）
  - 对策四：完善就业环境（4分），取消限制政策，保障平等就业
  - 对策五：提高就业能力（4分），加大培训（1分），创建评价体系（1分）
  - 对策六：建立覆盖城乡的公共就业服务体系（2分），整合资源（1分），提高保障水平（1分），全国信息查询系统（1分）
  - 结尾："农民兄弟、同城待遇"是应坚持的方向

### 两套答案异同比较

| 维度 | 答案A（解析一） | 答案B（解析二） |
|------|-----------------|-----------------|
| 第一题 | 概括性较强，用阶段特征+原因两段式 | 更细化，逐条列出赋分点，特征和原因都有明确分值 |
| 第二题 | 按时间线分三个阶段叙述工作进展 | 提炼三个核心特质（及时性、针对性、先进性），每点展开阐述 |
| 第三题 | 对策更注重制度层面的顶层设计 | 对策更具体、细化，每条对策带分值和子要点 |
| 整体风格 | 分析性强，语言简练 | 操作性更强，便于评分 |
| 亮点 | 第二题按时间轴线清晰展示政策演进脉络 | 第二题的"三性"提炼思路新颖，第三题带完整赋分说明 |

**评分参考价值**：答案B附带了详细赋分说明，对理解阅卷标准更有帮助；特别是第二题明确了"三性"为得分核心关键词，其他答案不超过10分。

## 2011年

### 考试概况
- **主题词**：职业病防治、尘肺病、劳动权益
- **材料主题**：以我国职业病高发为背景，结合张海超"开胸验肺"等典型案例，探讨职业病防治的困境与出路
- **材料数**：9则 | **题目数**：3题 | **满分**：100分
- **时限**：120分钟

### 给定材料概要
| 材料 | 核心内容 | 关键信息 |
|------|----------|----------|
| 材料1 | 职业病定义与历史演变 | 法定职业病四要件，工业革命以来职业卫生问题演变 |
| 材料2 | 我国职业病发病现状 | 2亿劳动者受危害，累计报告72万多例，2009年新增病例增319%，尘肺病占70.2% |
| 材料3 | 职业病危害因素多元化 | 电磁辐射等新危害，职业病因谱变化（广东：稻田皮炎→有机磷中毒→尘肺→中毒+噪音） |
| 材料4 | 联建科技正己烷中毒事件 | 苹果公司建议用正己烷替代酒精，134人受影响，违规未申报 |

| 材料8 | 国外经验 | 美国：普通医生可诊断、纳入医保；德国：法定强制保险、类职业病认定；巴西：自动认定 |

### 题目分析

  - 出路：传统模式难以为继，需走绿色发展之路

#### 第二题（20分）
- **题型**：综合分析（分析好处/意义）
- **作答要求**：根据材料5，分析说明发展循环经济有哪些好处；不超过200字
- **答题要点**：
  1. 对企业：推动技术研发和创新；节约资源、降低成本；扩大生产规模、提高竞争力；提升经济效益
  2. 对经济：提高资源利用率，协同效应明显，构建资源节约型社会；实现循环产业链条发展，推动产业结构调整和绿色发展
  3. 对环境：保护环境，实现清洁生产和废物资源化，减少污染和生态破坏，促进生态文明建设，构建环境友好型社会

#### 第三题（20分）
- **题型**：提出对策（建议类）
- **作答要求**：以G市交委工作人员身份，根据座谈会发言，就进一步完善公共自行车服务提出建议；不超过200字
- **答题要点**：
  1. 科学规划机动车道和非机动车道，保障骑车人安全
  2. 合理设置租赁点，在居民区、办公区增设和扩大租赁点；匹配对接点方便归还
  3. 加大政府资金补贴力度，创新合理营利渠道，做好车辆运营维修工作
  4. 加强管理，完善监督设备，设立监督平台杜绝偷盗；落实实名制，加大处罚力度
  5. 加强宣传教育，提升群众素质，确保锁好车后再离开

#### 第四题（50分）
- **题型**：议论文
- **题目**：《绿色发展》（命题作文）
- **总论点**：在经济发展与资源环境矛盾日益突出的情况下，既着眼当前更考虑长远，承担起推进绿色发展的重任
- **分论点框架**：
  1. 坚持规划先行，谋划绿色发展新方式——以主体功能区规划为例，不同区域不同开发方式，既要金山银山又要绿水青山
  2. 发挥创新作用，提供绿色发展新动力——以林全纸业创新技术实现大跨越为例
  3. 提高发展认识，培育绿色发展新理念——以生态消费理念为例，通过宣传教育引导公众
- **结尾引用**：党的十八大报告"建设生态文明，是关系人民福祉、关乎民族未来的长远大计"

## 跨年趋势分析

### 主题演变

| 年份 | 核心主题 | 选题领域 | 与当年热点的关联 |
|------|----------|----------|-----------------|
| 2009 | 地铁安全与地陷事故 | 城市建设/安全生产 | 2008年杭州地铁事故、广州地陷频发 |
| 2010 | 农民工用工短缺 | 社会民生/劳动就业 | 2009-2010经济复苏后的"用工荒" |
| 2011 | 职业病防治 | 公共卫生/劳动权益 | 张海超"开胸验肺"事件（2009年曝光） |
| 2012 | 政府与市场关系 | 经济体制/市场监管 | 广东"三打两建"（2012年启动） |
| 2013 | 社会管理创新 | 政府改革/社会治理 | 十八大后社会治理创新部署 |
| 2014 | 绿色发展/生态文明 | 生态环保/经济发展 | 十八大生态文明建设、雾霾治理 |

**演变规律**：从具体领域的安全事故（2009）→社会群体问题（2010-2011）→体制性经济问题（2012）→政府治理创新（2013）→宏观发展战略（2014），选题视野逐年扩大，从微观到宏观，从具体到抽象，与国家五年规划周期高度吻合。

### 题型演变

| 年份 | 题量 | 题型结构 | 分值分布 |
|------|------|----------|----------|
| 2009 | 3题 | 概括+简报+策论 | 25+25+50 |
| 2010 | 3题 | 概括(分阶段)+分析(提炼特点)+策论 | 30+20+50 |
| 2011 | 3题 | 概括(特点)+分析(原因)+策论 | 30+20+50 |
| 2012 | 3题 | 概括(构成)+对比分析+议论文(自拟) | 20+30+50 |
| 2013 | 3题 | 概括(做法)+概括(特点)+议论文(命题) | 20+30+50 |
| 2014 | **4题** | 概括+分析(好处)+对策+议论文(命题) | 10+20+20+50 |

**关键变化**：
1. **2014年首次出现4题模式**，增加了一道10分的小概括题，总时间从120分钟延长至阅读30分钟+作答90分钟
2. **贯彻执行题（简报）仅出现在2009年**，之后年份未再出现
3. **综合分析题型多样化**：2010年提炼特点、2011年分析原因、2012年对比分析、2014年分析好处，每年分析角度不同
4. **大作文命题方式变化**：2012年首次要求自拟题目（限定角度），2013-2014年均为命题作文
5. **对策题复苏**：2009年的概括教训包含对策思路，2014年明确要求"提出建议"

### 转型期申论特征总结

**与早期（2003-2008年）的差异**：

1. **材料质量提升**：2009-2014年的材料信息密度更高、论证更充分，既有宏观背景又有微观案例，既有国内实践又有国外经验借鉴，材料间的逻辑关联性更强。

2. **作答要求更精细**：
   - 字数要求从早期的"不超过XXX字"发展为更明确的规定
   - 要求从"概括准确"升级为"准确、全面、有条理""分析全面、透彻"
   - 2010年第二套答案首次出现详细的赋分说明，为考生理解评分标准提供了明确指引

3. **对策导向的变化**：从早期偏重"问题+对策"的简单模式，发展为需要先分析提炼再提出对策（2010年）、先对比再论述（2012年）的复合型思维模式。

4. **大作文能力要求升级**：
   - 2012年要求"从处理好政府与市场关系的角度"自拟题目，增加了提炼材料核心议题的难度
   - 2013-2014年的命题作文（《管理与服务》《绿色发展》）要求考生在给定题目下自主建构论述框架
   - 范文论证手法成熟，引入"意义段"（2014年第二段阐述绿色发展意义）、引用政策表述、使用典型案例作为论据

5. **广东省情深度融合**：
   - 2010年：广东"双转移"战略
   - 2012年：广东"三打两建"
   - 2013年：D市（东莞）流动人口管理、湛江/汕头网格化、广州社会组织改革
   - 2014年：广东省主体功能区规划、三水光伏产业、韶关绿色转型
   - 考生需对广东省重大政策有一定了解

6. **2014年参考答案的典型特征**：
   - 结构高度规范化：第一题100字内高度凝练（问题+出路），第二题分维度（企业+经济+环境），第三题分条列出（5条建议），第四题"意义段+分论点"模式成熟
   - 结合材料程度深：每道题都有明确的材料来源限定
   - 实操性增强：第三题（公共自行车建议）的场景化程度高，要求代入具体身份（交委工作人员）作答

## 一、第四章 增长率（第1-20页）

### 易错概念辨析

#### 1. 百分数与百分点
- **百分数**：表示两个量的比例关系，用**除法**计算
- **百分点**：表示百分数的变化，用**加减法**计算
- 坑点：官方表述用"百分点"，部分考官写作"%"实际指百分点

#### 2. 增长率与倍数
- A是B的多少倍：`A/B`
- A比B增长/多多少倍：`r = (A-B)/B = A/B - 1`
- **公式**：`增长倍数 = r`，`倍数 = r + 1`

#### 3. 成数与翻番
- 成数：几成 = 十分之几（如30% = 三成）
- 翻番：翻n番为原来的 `2^n` 倍

#### 4. 增幅、降幅、变化幅度
- **增长率/增幅/增速**：带正负号比大小
- **降幅**：必须为负，比较**绝对值**
- **变化幅度**：可正可负，比较**绝对值**

### 核心公式

**增长率基本公式：**
```
r = (现期 - 基期) / 基期 = 增长量 / 基期
```

### 1. 增长率计算

#### （1）百分点计算

**核心方法：**
- **"增速"** → 高减低加（带正负号计算）
- **"降幅"** → 先加负号，再对绝对值高减低加

**典型题**

**例1（2024四川）** 2022年全国软件业利润总额12648亿元，同比增长5.7%，增速较上年同期回落1.9个百分点。软件业务出口额524.1亿美元，同比增长3.0%，增速较上年同期回落5.8个百分点。问2021年利润总额同比增速比出口额同比增速：
> **解析**：
> 利润2022增速5.7%，较上年回落1.9个百分点 → 2021增速 = 5.7% + 1.9% = 7.6%
> 出口2022增速3.0%，较上年回落5.8个百分点 → 2021增速 = 3.0% + 5.8% = 8.8%
> 差值 = 7.6% - 8.8% = -1.2% → 低1.2个百分点 → **A**

**例2（2024江苏）** 2023年6月末江苏省、设区市国资委监管企业资产负债率63.6%，较上年末上升0.8个百分点。其中省国资委监管企业64.8%，下降0.1个百分点。问2022年末省国资委监管企业比省、设区市国资委监管企业高：
> **解析**：
> 省、设区市2023末63.6%，较上年末上升0.8% → 2022末 = 63.6% - 0.8% = 62.8%
> 省国资委2023末64.8%，下降0.1% → 2022末 = 64.8% + 0.1% = 64.9%
> 差值 = 64.9% - 62.8% = 2.1个百分点 → **C**

**例3（2024浙江）** 2023年第14周全市ILI%为5.98%，比2022年同期低7.56个百分点，比2021年同期低0.66个百分点……问2021年第14周从大到小排序：
> **解析**：逆向百分点计算——已知2023值和与2022、2021的差值，反推2021各值后排序 → **B**

#### （2）变形：百分点计算

**例4（2023广东事业单位）** 广东全社会就业人员年均增长率1.5%，城镇就业人员年均增长率高于全社会1.2个百分点。问2012-2021年广东城镇就业人员平均增长率：
> **解析**：城镇年均增长率 = 1.5% + 1.2% = 2.7%，在2%-3%之间 → **C**

#### （3）具体值计算

**① 给现期和基期：** `r = (现期 - 基期) / 基期`

**例5（2024联考）** 2014-2022年全国亿元以上食品、饮料及烟酒市场成交额同比增量最高的年份，其茶叶市场成交额同比增速为：
> **解析**：先找增量最大年份（柱状图数据），再计算该年茶叶市场增速 → **C（15%-20%）**

**例6（2024联考）** 2022年中国、巴西、俄罗斯、印度、南非五国创新指数之和同比：
> **解析**：加总五国2022和2021数据，代入增速公式 → **A（增长约3.2%）**

**② 给现期、增长量：** `r = 增长量 / (现期 - 增长量)`

**例8（2024联考）** 截至2022年12月，我国城镇网民规模约7.59亿（较2021年12月增量数据需从材料提取），同比增速约为：
> **解析**：城镇网民增量需计算（总增量-农村增量），r = 增量 / (现期-增量) → **A（1.2%）**

**例9（2023重庆事业单位）** 2022年移动通信基站总数1083万个，全年净增87万个，同比增速约为：
> **解析**：r = 87 / (1083-87) = 87/996 ≈ 8.7% → **C（9%）**

**③ 变形题：给基期量、增长量：** `r = 增长量 / 基期量`

**例10（2020联考）** 2016年语言类大学平均薪酬排名第二的院校，其毕业生平均薪酬同比增长约为：
> **解析**：从表格中找到排名第二院校的基期和增长量，直接相除 → **A（18%）**

#### （2）给现期、增长量

**方法：** 增长率 = 增长量 / (现期 - 增长量)，转化为分数比较

**例20（2024国考）** S省12315投诉55.6万（+14.0万）、举报26.3万（+8.9万）、咨询138.5万（+16.0万），按2022年同比增速从高到低排序：

## 二、第五章 增长量（第21-39页）

### 1. 增长量计算

#### （1）给现期、基期

**公式：** `增长量 = 现期 - 基期`

**例1（2023广东）** 2021年广东省财政科学技术支出比上年提高多少亿元：
> **解析**：直接读取2021和2020数据相减 → **C（27.03亿元）**

**例3（2022广东）** 2020年全国基层群众性自治组织较上年减少多少万个：
> **解析**：现期-基期，结果为负取绝对值 → **B（2.8万个）**

#### （2）给现期、r（百化分）

**核心公式：**
```
增长量 = 现期 / (1+r) × r

当 r = 1/n 时（百化分）：
  增长量 ≈ 现期 / (n+1)   （r>0）
  减少量 ≈ 现期 / (n-1)   （r<0）
```

**① 直接百化分**

**例5（2024联考）** 2023年1-5月民营企业出口汽车约10.6万辆，同比增长64.3%，同比增加了多少万辆：
> **解析**：64.3% ≈ 1/1.555，更精确：64.3% ≈ 5/7.78，实际使用百化分：
> 64.3% ≈ 1/1.56，增长量 ≈ 10.6/(1+1.56) ≈ 4.1 → **B（4万辆）**

**例6（2024联考）** 2023年11月J省深圳证券交易所债券当期成交额同比增量：
> **解析**：找到现期和r，百化分后计算 → 答案见选项

**例8（2023联考）** 2022年1-7月全国城乡居民生活用电量7586亿千瓦时，同比增长12.5%，比2021年同期约多：
> **解析**：12.5% = 1/8，增长量 = 7586/(8+1) = 7586/9 ≈ 843 → **B（843亿千瓦时）**

**例9（2023联考）** 2021年全国城市供水管道长度105.99万公里，同比增长5.26%，比2020年增长约：
> **解析**：5.26% ≈ 1/19，增长量 ≈ 105.99/20 ≈ 5.3 → **B（5.3万公里）**

**② 放缩**

当r不是标准百化分数值时，寻找近似的百化分后微调。

**例10（2020联考）** 2019年6月全国发行地方政府债券8996亿元，同比增长68.37%，比2018年6月多约：
> **解析**：68.37% ≈ 2/3（即1/1.5），增长量 = 8996/(1+2/3) = 8996×3/5 ≈ 5398 → 精确选 **C（3653亿元）** ，此处放缩需更精确

**③ 取中**

当r介于两个百化分之间时，取中间值估算。

**例11（2022北京）** 2019年A市专利密集型产业实现增加值6918.8亿元，比上年增长9.6%，增长约多少亿元：
> **解析**：9.6%介于9.1%(1/11)和10%(1/10)之间，取中≈1/10.4
> 增长量 ≈ 6918.8/11.4 ≈ 607 → **C（606亿元）**

**例12（2023联考）** 2021年我国境内专利授权数比上一年增加：
> **解析**：同取中法估算 → **C（85万件）**

**④ |r|<5%（增长量 ≈ 现期量 × r）**

当增长率小于5%时，忽略分母中的r，直接用现期×r估算。

**例13（2024江苏）** 2023年上半年省国资委监管企业营业收入2091亿元，增长4%，同比增加：
> **解析**：|r|<5%，增长量 ≈ 2091×4% ≈ 83.6 → **B（80亿元）**

**例14（2024广东事业单位）** 2022年全国海水产品产量3459.53万吨，同比增长2.1%，同比增长：
> **解析**：|r|<5%，增长量 ≈ 3459.53×2.1% ≈ 72.7 → **C（60-80万吨）**

**⑤ r < 0**

**例15（2024联考）** 2022年十个品种再生资源回收总金额13140.6亿元，同比下降4.05%，同比下降了约：
> **解析**：4.05% ≈ 1/25，减少量 = 13140.6/(25-1) = 13140.6/24 ≈ 547.5 → **C（555亿元）**

#### （3）增长量的倍数

**方法：** 分别计算两个增长量再相除。

**例17（2024国考）** 2019年蜂窝物联网终端用户数同比增量约是移动电话用户数同比增量的多少倍：
> **解析**：用百化分分别计算两者增长量后相除 → **A（7倍）**

#### （4）增长量变形（增长1%的值）

**公式：** `增长1%的值 = 基期 × 1%`

**例19（2020山东）** 2017年J省海洋生产总值7217亿元，增长9.2%，每增长1个百分点约增加：
> **解析**：基期 = 7217/(1+9.2%) ≈ 6609，1% = 6609×1% ≈ 66亿元 → **A（66亿元）**

#### （5）拉动增长率与增长贡献率

**公式：**
```
拉动增长率 = 部分增长量 / 整体基期量
增长贡献率 = 部分增长量 / 整体增长量
```

**例20（2018陕西）** 2017年移动数据及互联网业务收入拉动电信业务收入增长约多少个百分点：
> **解析**：移动数据收入5489亿元，增长26.7%，增量 ≈ 5489/4.75 ≈ 1156
> 电信业务收入12620亿元，增长6.4%，基期 ≈ 11861
> 拉动 = 1156/11861 ≈ 9.7% → **C（9.7个百分点）**

**例21（2024江苏事业单位）** 2023年江苏第三产业对全省经济增长的贡献率是：
> **解析**：三产增量/全省增量 = (66236.7-三产基期)/(128222.2-全省基期) → **B（45.7%）**

#### （6）年均增长量

**公式：** `年均增长量 = (末期 - 基期) / 年份差`

- **非限定**：年份差 = 末年 - 基年
- **五年规划**：基期为规划前一年，年份差固定为5
- **江苏前推**：基期再前推一年（如2011-2018，基期算到2010）

**例22（2024广东）** 2013-2022年中国对外直接投资流量年均增加约：
> **解析**：非限定，(2022值-2013值)/9 → **B（61.4亿美元）**

**例23（2024国考）** 以2013年为基期，2013-2022年京津冀地区生产总值年均约增加多少万亿元：
> **解析**：(10.0-10.0/1.8)/9 ≈ 0.5 → **D（0.5万亿元）**

**例25（2022江苏）** "十三五"时期（2016-2020），我国住房公积金实缴余额年均增量：
> **解析**：五年规划，基期为2015年，(2020值-2015值)/5 → **B（0.43万亿元）**

**例26（2020江苏）** 2011-2018年全国文物机构从业人员年均增加：
> **解析**：江苏前推，基期前推到2010年，(2018值-2010值)/8 → **A（7516人）**

## 三、第六章 间隔增长率（第41-49页）

### 1. 间隔增长率

**公式：** `r间隔 = r1 + r2 + r1 × r2`
> 其中r1是2022年增速，r2是2021年增速（通常通过百分点计算反推）

**例1（2024四川）** 2022年全国软件业务收入108126亿元，同比增11.2%，增速较上年回落6.5个百分点。问2022年比2020年增长：
> **解析**：
> r1 = 11.2%，r2 = 11.2%+6.5% = 17.7%
> r间隔 = 11.2% + 17.7% + 11.2%×17.7% ≈ 28.9% + 1.98% ≈ 30.9% → **D（31%）**

**例2（2024联考）** 2022年全国新发理财产品2.94万只，同比下降38.23%，降幅比上年扩大7.22个百分点。与2020年相比约：
> **解析**：
> r1 = -38.23%，r2 = -(38.23%-7.22%) = -31.01%
> r间隔 = -38.23% -31.01% + 38.23%×31.01% ≈ -69.24% + 11.85% ≈ -57.4% → **B（下降57%）**

### 2. 间隔倍数

**公式：** `间隔倍数 = r间隔 + 1`

**例3（2024全国事业单位联考）** 2023年2月全国旅游客车销量约是2021年2月的多少倍：
> **解析**：先算间隔增长率，再加1 → **B（1.5倍）**

### 3. 间隔基期

**公式：** `间隔基期 = 现期 / (1 + r间隔)`

**例4（2020四川下）** 2015年城镇居民人均可支配收入31195元，增长8.2%，增长率比2014年下降0.8个百分点。问2013年城镇居民人均可支配收入：
> **解析**：
> r1=8.2%，r2=8.2%+0.8%=9.0%
> r间隔 = 8.2%+9.0%+8.2%×9.0% ≈ 17.2%+0.74% ≈ 17.94%
> 2013年 = 31195/(1+17.94%) ≈ 26450 → **C（2.6万元）**

**例5（2023福建事业单位）** 2022年M地区汽车行业主营业务收入83345.25亿元，增长13.79%，比2021年上升9.06个百分点。问2020年收入：
> **解析**：
> r1=13.79%，r2=13.79%-9.06%=4.73%
> r间隔 = 13.79%+4.73%+13.79%×4.73% ≈ 19.17%
> 2020年 = 83345.25/(1+19.17%) ≈ 69937 → **A（69937亿元）**

### 4. 间隔增长量

**方法：** 先算间隔增长率，再用百化分算增长量。

**例6（2019辽宁）** 2018年规模以上工业主营业务利润总额比2016年增长约多少亿元：
> **解析**：r间隔计算→百化分求增长量 → **D（928亿元）**

### 5. 间隔现期

**方法：** `现期 = 基期 × (1+r年均)^n`，用年均增速估算间隔多年的现期。

**例7（2021江苏）** 按2001-2019年年均增速估算，2021年全国居民人均医疗保健支出介于：
> **解析**：已知2019年值和2000-2019年均增速14.7%，估算2021年值 → **C（2420-2470元）**

### 6. 间隔多年增长率

**方法：** 已知年均增长率r，求n年的总增长率：`R总 ≈ r × n + C(n,2) × r²`（或直接用 `(1+r)^n - 1`）

**例8（2017联考）** 2015年服务业投资额31.19万亿元，2012-2015年均增长15.9%。2015年相当于2012年的增长率约为：
> **解析**：3年，R总 = (1+15.9%)³ - 1，或用估算：15.9%×3 + 3×15.9%² ≈ 47.7%+7.6% ≈ 55.3% → **B（56%）**

**例9（2020广东）** 2014-2019年我国农民工规模增加了约：
> **解析**：5年总增长率，用间隔思维 → **C（6.2%）**

### 7. 间隔的逆向运用

**核心方法：** 已知r间隔和r1反推r2：`r2 = (r间隔 - r1) / (1 + r1)`

**例10（2023国考）** 2021年服装出口1702.8亿美元，同比增长24.0%，较2019年增长16.0%。问2020年服装出口比2019年：
> **解析**：r间隔=16%，r1=24%，求r2
> 16% = 24% + r2 + 24%×r2
> r2 = (16%-24%)/(1+24%) = -8%/1.24 ≈ -6.45% → **D（下降不到10%）**

**例11（2022江苏）** 2021年1-7月原油产量11561万吨，同比增2.4%，比2019年同期增3.9%。问2020年1-7月同比增速：
> **解析**：r间隔=3.9%，r1=2.4%，r2 = (3.9%-2.4%)/(1+2.4%) = 1.5%/1.024 ≈ 1.46% → **A（1.46%）**

**例12（2023深圳）** 2021年1-2月商品房销售面积同比增1.05倍，比2019年1-2月增23.3%。问2020年1-2月同比增速：
> **解析**：r间隔=23.3%，r1=105%，r2 = (23.3%-105%)/(1+105%) = -81.7%/2.05 ≈ -39.9% → **A（-40.0%）**

**例14（2018国考）** 2017年2月末手持船舶订单同比下降22.6%，比2016年末下降7.6%。问2016年末较同年2月末：
> **解析**：r间隔=-7.6%，r1=-22.6%（2017.2较2016.2），求r2（2016.12较2016.2）
> -7.6% = -22.6% + r2 + (-22.6%)×r2
> r2 = (-7.6%+22.6%)/(1-22.6%) = 15%/0.774 ≈ 19.4%，但实际需注意方向
> 2016年末手持订单较同年2月末是降低的 → 重新计算得 **A（降低16.2%）**

### 8. 间隔逆运用与年均增长率的应用

**例15（2023浙江）** 2021年上半年S市战略性新兴产业总产值同比增19.6%，两年平均增长12.3%。问2020年上半年同比增速：
> **解析**：两年平均增速即r间隔的几何平均
> (1+12.3%)² = 1+19.6%+r2+19.6%×r2 → 解出r2 → **A（不到7%）**

**例16-17（2024四川）** 两年平均增长（13.3%、16.0%）的逆向运用，反推2020年1-2月增速：
> **解析**：同上述方法 → **B（仅②同比正增长）、A（吃类、穿类、用类）**

## 四、第七章 乘积增长率（第51-59页）

### 1. 总额 = 单价 × 数量

**公式：** 若 A = B × C，则 `rA = rB + rC + rB × rC`

> 总价 = 单价 × 产量，r总价 = r单价 + r产量 + r单价×r产量
> 从材料中找到零售价格增速和产量增速后代入公式 → **C（17.6%）**

**例3（2023重庆事业单位）** 2022年6月规模以上工业原油产量的环比增长率约为：
> **解析**：利用产量与相关指标的乘积关系反推 → **A（0.6%）**

### 棉花专题——材料的特色考法

**核心方法：** 总产量 = 播种面积 × 单产，求总产量增速时用乘积增长率公式。

**例4** 2017年长江流域棉区棉花总产量同比约下降：
> **解析**：面积增-14.9%，单产增-3.6%
> r总 = -14.9% - 3.6% + (-14.9%)×(-3.6%) ≈ -18.5% + 0.54% ≈ -18% → **D（18%）**

**例5** 2017年新疆棉区棉花总产量同比约：
> **解析**：面积增8.7%，单产增4.4%
> r总 = 8.7%+4.4%+8.7%×4.4% ≈ 13.1%+0.38% ≈ 13.5% → **B（13.5%）**

**例6** 2017年黄河流域棉区棉花总产量同比约：
> **解析**：面积增-24.3%，单产增2.1%
> r总 = -24.3%+2.1%+(-24.3%)×2.1% ≈ -22.2%-0.51% ≈ -22.7% → **B（-22.7%）**

**例7** 2016年黄河流域棉区棉花播种面积约为：
> **解析**：基期 = 现期/(1+r) = 现期/(1-24.3%) → 从材料找现期面积计算 → **C（885.2千公顷）**

### 2. 部分 = 整体 × 比重

## 五、第八章 年均增长率（第61-65页）

### 1. 年均增长率的比较

**核心方法：** 比较 `末期/基期` 的倍数关系，年份相同时倍数大的年均增速大。

**例1（2024广东）** 判断2019-2021年东部地区乡村人均社会资本投入年均增长率是否高于中部和西部地区：
> **解析**：比较 `2021/2019` 的倍数 → **B（错误）**

**例2（2022四川）** 按2016-2019年业务年均增速从高到低排列同城、异地、国际/港澳台快递：
> **解析**：比较 `2019量/2016量` 的倍数 → **D（③①②）**

**例3（2023国考）** 2016-2021年集成电路产量年均增速从高到低排列甘肃、广东、上海、浙江：
> **解析**：比较 `2021值/2016值` 倍数 → **C（①④②③）**

### 2. 年均增长率的计算

#### （1）r < 10%（估算、间隔思维解题）

**方法：** 年均增长率 ≈ (末期/基期 - 1)/n，或用间隔思维 `(1+r)^n ≈ 1 + n×r`

**例4（2021广东）** 2016-2020年G省社会消费品零售总额年均增长率约为：
> **解析**：4年，r < 10%，用 `(1+r)^4 = 末期/基期` 估算 → **B（5%）**

#### （2）r > 10%

**方法：** 代入选项验证，或 `(1+r)^n = 末期/基期` 用居中代入法。

**例5（2024广东）** 2019-2021年全国县域农业农村信息化建设县均社会资本投入的年均增长率约为：
> **解析**：2年，(1+r)² = 2021值/2019值，代入选项 → **C（32%）**

**例6（2022广东）** 2016-2018年我国国际重要湿地面积的年均增长率约为：
> **解析**：2年，(1+r)² = 2018值/2016值 → **B（30%）**

**例7（2019联考）** 2017年马拉松产业总规模700亿元，目标2020年1200亿元，每年平均增长：
> **解析**：3年，(1+r)³ = 1200/700 ≈ 1.714
> 代入验证：1.2³=1.728，r≈20% → **C（20%）**

**例8（2018上海）** 2012-2016年我国单银幕总票房平均每年较上年增长约：
> **解析**：4年，(1+r)⁴ = 末期/基期 → **B（28%）**

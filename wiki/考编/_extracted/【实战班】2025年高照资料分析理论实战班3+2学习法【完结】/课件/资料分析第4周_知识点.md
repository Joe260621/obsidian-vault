# 第4周：平均数·倍数·A和非A思想·综合分析

> 核心掌握平均数与倍数的六种题型公式、A和非A思想的正反向思维贯穿，以及综合分析的"能推出/不能推出"判断技巧。

---

## 一、平均数（第十一章）

### 1. 现期平均数

#### （1）平均数计算

**核心方法：** 后÷前法 + 单位辅助 + 常识判定
- 关键字：均、每、单位面积、单价
- 公式：平均数 = 总数/个数 = A/B
- 判定分母三途径：
  1. 后÷前（标准问法："人均收入"→收入/人数）
  2. 单位辅助（km/h→路程/时间）
  3. 常识定义（单位面积产量 = 产量/面积）

**例1（2024全国事业单位联考）** 2023年前三季度，S省1627家文化企业实现营业收入862.76亿元，求平均每家营收范围。
> 862.76 ÷ 1627 ≈ 0.53亿元 → **A.不到0.5亿元**

**例2（2024广东）** G省法律援助案件195779件，帮助群众212993人，求平均每件帮助人数。
> 212993 ÷ 195779 ≈ 1.088 → **A.小于1.1人**

**例3（2024江苏）** 住院费用282.6万人次，减少垫付354.3亿元，求次均减少垫付金额。
> 354.3 ÷ 282.6 ≈ 1.254 → **C.1.25万元**

#### （2）平均数和某个定值比较

**要点：**
- 注意单位转换坑点（千万/亿、万吨/吨等）
- 把÷转化为×进行精确比较（如判断A/B是否＞3 → A＞3B）

**例4（2024联考）** 判断2013-2022年全国亿元以上食品饮料市场成交额超过烟酒市场3倍以上的年份个数。
> A＞3B逐个验证 → **B.7个**

**例5（2021山东）** 2019年棉花种植面积前7省区中，单产超过1吨/公顷的有几个？
> 单位转换为吨/公顷，逐个计算比较 → **A.5**

#### （3）日均、月均

**核心结论：** 某个月份＞前几个月的平均值 ⇔ 该月份＞所有月份的平均值
- 推导：若12月产值 > 1-11月平均值，则12月产值 > 1-12月平均值
- 既可用"12月 > 全年/12"判定，也可用"12月×12 > 全年总额"判定

**例6（2024浙江）** 2022年1-4月，福利彩票月均销售额约多少亿元？
> 各月相加÷4，或（1-4月总额）÷4 → **C.120**

**例7（2024国考）** 判断2022年12月工业机器人和服务机器人是否高于1-11月月均水平。
> 12月 > (累计-12月)/11 等价于 12月×12 > 累计全年 → **C.仅①高于**

#### （4）多个数求平均数：削峰填谷

**方法：** 削峰填谷 + 高位叠加
- 选一个基准值，各数偏差正负抵消后取平均
- 高位叠加：先加高位再加低位

**例8（2024联考）** 2023年上半年J省深圳证券交易所债券平均成交额。
> 削峰填谷（选基准15百亿元，偏差求和后÷6）→ **A.不到15百亿元**

**例9（2024联考）** 2019-2022年中国数码锂电池年均出货量。
> 削峰填谷 → **C.47-49GWh之间**

#### （5）求整体或数量

**例10（2020四川上）** 已知人均和总数，反推整体（户数/人数）。
> 考查平均数反推总量公式：整体 = 平均数 × 个数

#### （6）平均数比例类比

**要点：** 和比重比例类比同思路。求平均数但无完整分母时，可利用存在相同分母的平均数做比例类比。

**例11（2024联考）** 2022年全国每万人拥有的体育场地数量。
> 利用已知总量和总人口比例类比计算 → **B.29.9个**

### 2. 基期平均数

**识别：** 过去时间 + 平均

**公式：**
1. 给现期量+增长率：基期平均数 = A/B × (1+b)/(1+a)
2. 给现期量+增长量：基期平均数 = (现期A-增量A)/(现期B-增量B)

**例12（2019浙江）** 2017年内燃机累计销量5645.38万台（+4.11%），累计功率266879.47万千瓦（+9.15%），求2016年平均功率。
> 基期平均数 = (266879.47/5645.38) × (1+4.11%)/(1+9.15%) ≈ 47.28 × 0.954 ≈ 45KW → **B.45千瓦**

### 3. 两期平均数

#### （1）两期平均数比较

**判定法则：**
- a > b → 平均数上升
- a < b → 平均数下降
- a = b → 平均数不变
（a为分子增长率，b为分母增长率）

**例13（2024江苏）** 判断哪个项目的次均垫付金额环比有所增加。
> 分别比较各项目人次与金额的环比增长率：人次增速 < 金额增速则次均金额上升 → **D.门诊慢特病**

#### （2）平均数增长量

**题型特征：** 平均数问法 + 增长 + 具体单位（元/人/吨等）

**公式：** 增长量 = A/B - A/B × (1+b)/(1+a) = A/B × (a-b)/(1+a)

**速算技巧：** 多步除法，截位列式，约分看选项

**例14（2024江苏事业单位）** 2023年国内出游48.9亿人次（+93.3%），总花费49133亿元（+140.3%），求人均花费增长量。
> 增长量 = (49133/48.9) × (140.3%-93.3%)/(1+140.3%) ≈ 1005 × 0.196 ≈ 197元 → **A.197元**

#### （3）平均数增长率

**识别：** 平均数 + 增长……%

**公式：** r = (a - b) / (1 + b)
- 注：a为分子的增长率，b为分母的增长率

**做题步骤：**
1. 根据平均数确定分子和分母
2. 代入公式 r = (a-b)/(1+b)
3. 看选项速算

**特殊情况：** b=0时，r = a

**例15（2024联考）** 天津出口汽车17.2万辆（+29.5%），总价值100.1亿元（+40.2%），求均价同比上涨。
> 均价增长率 = (40.2%-29.5%)/(1+29.5%) = 10.7%/1.295 ≈ 8.3% → **A.8%**

**例16（2024联考）** 新疆有进出口实绩企业1535家（+12%），进出口值（总体增长率需从材料提取），求平均每家企业进出口值增长率。
> 利用进出总值增长率与企业数量增长率代入公式 → **A.61%**

**例17（2020国考）** 2018年中国平均每块集成电路出口单价比上年。
> r = (a-b)/(1+b)，需从图表提取a和b → **D.上升了30%以内**

### 4. 变形题

#### （1）连续年份平均数比较转化为两期平均数比较

**方法：** 连续年份平均数的同比增长判定，转化为比较逐年a和b的大小关系即可。

**例18（2024国考）** 2018-2022年马铃薯单位面积产量同比增长的年份个数。
> 逐年比较产量增长率与面积增长率 → **B.3**

**例19（2020四川下）** 2014-2017年出境旅游平均人次花费最多的年份。
> 旅游花费增速a与出境人次增速b，求a-b最大年 → **C.2016**

#### （2）间隔平均数增长率

**方法：** 先求间隔增长率，再代入平均数增长率公式。

**例20（2020四川上）** 2017年S市服务业小微样本企业平均每万元资产实现营业收入比2015年增长/下降多少。
> 先求营业收入的间隔增长率，再求资产的间隔增长率，最后代入平均数增长率公式 → **A.增长了不到5%**

#### （3）平均数增长率和乘积增长率

**量的关系：** 平均数 × 个数 = 总数
**率的关系：** 平均数增长率与乘积增长率相互推导的逆运算。

**例21（2021上海）** 通过人均上网时长和总时长的关系反推网民数量的增长率。
> 利用乘积增长率公式，已知总时长增长率和人均时长增长率，求网民数量增长率 → **D.增加了5.4%**

## 二、倍数（第十二章）

### 核心公式

**判定关键字：** "是几倍""多几倍""超过几倍"
- A是B的……倍：A/B
- A比B增长/多/高/超出……倍：A/B - 1
- A超过B的n倍：A > B × n

### 1. 现期倍数

**例1（2024广东）** 东部社会资本投入562.4亿元，中西部之和193.8+198.4=392.2亿元，求东部是中西部的倍数。
> 562.4 ÷ 392.2 ≈ 1.43 → **A.1.43**

**例2（2024国考）** 蜂窝物联网终端用户同比增量是移动电话用户同比增量的多少倍。
> 从图中读取增量数据后直接相除 → **A.7**

**例3（2024国考）** 中部六省中型灌区新增恢复灌溉面积是东北三省的多少倍。
> 读出数据后直接做除法 → **A.4.5~5倍之间**

**例4（2024国考）** 2023年1~2月服务机器人产量约是工业机器人的多少倍。
> 利用1-3月累计减3月得到1-2月数据，再求倍数 → **B.14.0**

**例5（2023联考）** 城市人均日生活用水量最高的省份是用水量中位数省份的多少倍。
> 找出最大和中位数后直接相除 → **B.1.88倍**

**例6（2024国考）** 中型灌区改善灌溉面积与新增恢复灌溉面积比值最大的中部省份。
> 分别计算各省比值，选最大 → **D.江西**

### 2. 增长倍数

**例7（2024国考）** 2018-2022年全国电化学储能电站年末总能量同比增长100%以上的年份个数。
> 逐年比较，当年/上年>2即为同比增长>100% → **B.2**

**例8（2022江苏）** 2021年上半年我国原油进口量比生产量多多少倍。
> 先求上半年进口量和生产量，再用（进口-生产）/生产 → **A.1.6倍**

**例9（2023福建事业单位）** 与2013年相比，2020年我国网民规模约增加了多少倍。
> （2020年-2013年）/2013年 → **A.0.6**

### 3. 比较某定值倍数

**例10（2022广东）** 国内航线条数比国际航线条数多4倍以上的年份个数。
> 判定条件：国内/国际 > 5（多4倍即A/B-1>4→A/B>5） → **B.2**

**例11（2019国考）** 支付交易类钓鱼网站数量超过金融证券类2倍的月份个数。
> 逐月比较，支付交易类 > 金融证券类 × 2 → **B.6**

### 4. 指桑骂槐类倍数

**题型特征：** 主语省略，前面是重点，省略部分在后面才出现。

**例12（2022四川）** 2019年平均工资最高行业的专业技术人员平均工资是生产制造及有关人员的多少倍。
> 先锁定平均工资最高的行业，再求其下两类人员的倍数 → **D.1.9**

### 5. 平均数倍数、比重倍数、增长量倍数（多步除法）

**长句子专项要点：**
- 主语省略型：前为被省略主语
- 计算方法：多步除法，截位列式
- **技巧：** 分子分母存在整体部分关系时，直接用比例更快

**例13（2024四川）** 东部地区软件业务收入同比增量约是西部地区的多少倍。
> 增量 = 现期×r/(1+r)，东部88663×10.6%/1.106，西部11574×14.3%/1.143 → **C.5**

**例14（2023浙江）** S市批发和零售业零售额同比增量约是住宿和餐饮业的多少倍。
> 增量倍数 = (8287.13×28.4%/1.284)/(761.31×54%/1.54) → **C.7**

**例15（2021国考）** 平均每个综合类直排海污染物排口排放污水量约是工业类的多少倍。
> 综合类平均/工业类平均 = (综合总量/综合个数)/(工业总量/工业个数) → **C.1.4**

**例16（2021国考）** 全国平均每个已竣工残疾人综合服务设施建设规模约是康复设施的多少倍。
> (584.5/2341)/(414.2/1006) ≈ 0.6 → **A.0.6**

**例17（2022四川下）** 平均每个各级文旅所属艺术表演团体的从业人员约是全国所有艺术表演团体的多少倍。
> (10.75/2060)/(43.69/17581) → **D.2.1**

**例18（2020北京）** "一带一路"沿线国家人均GDP约是全球水平的多少倍。
> 利用GDP占比和人口占比的比例关系：（16%）/（43.3%）≈ 0.37 → **D.0.4**

**例19（2018江苏）** 苏北大专及以上学历农业生产经营人员是苏中地区的多少倍。
> 利用各部分占比或比例关系速算 → **C.2.6**

### 6. 基期倍数

**识别：** 问题时间在材料时间之前 + 倍

**公式：** 基期倍数 = A/B × (1+b)/(1+a)

**例20（2024联考）** 2021年铁路交通固定投资额是民航的多少倍。
> 观察时间，若材料给2022年数据，则用基期倍数公式 → **A.5.1倍**

**例21（2024联考）** 2022年前三季度文化服务业营业收入约是文化批发和零售业的多少倍。
> 材料给2023年前三季度数据，基期倍数公式 → **B.2.5~3倍**

## 三、A和非A思想（第十三章）

**核心思想：** A + 非A = 整体，正反向思维的贯穿运用，涵盖比重、增长量、增长率、倍数、乘积增长率等多个领域。

### 1. A和非A思想在比重中的运用

#### （1）占比大于某个比例

**方法：** 转化为各部分的倍数关系（份数思想）
- 如判断是否超过50% → 判断A是否 > 非A
- 超过75% → A > 3×非A
- 超过25% → A > 非A/3

**例1（2021山东）** 2014-2019年4G基站数占移动电话基站总量一半以上的年份个数。
> 逐年判断4G > 非4G即占一半以上 → **C.3**

**例2（2022国考）** 2020年下半年环保举报受理率超过75%的月份个数。
> 受理 > 3×不受理（即受理量 > 不受理量的3倍）→ **C.5**

#### （2）占比最大

**结论：** 占比越大，A和非A的倍数越悬殊（类比"风浪越大，鱼越贵"）

**例3（2024广东事业单位）** 2018-2021年广东省海水养殖产量占海水产品产量比重最高的年份。
> A/非A最大即占比最高 → **D.2021年**

**例4（2022国考）** 2020年1-4月微信举报占比最高的月份。
> 微信举报/（非微信举报）比值最大 → **B.2月**

#### （3）求占比

**方法：** 利用整体 = A + 非A的关系，求各部分量或占比

**例5（2024国考）** 截至2021年末全国累计投运电化学储能电站中小型电站数量占比。
> 小型 = 总数 - 大型 - 中型，再用小型/总数 → **B.35%**

**例6（2023浙江）** 2020年上半年S市新建非住宅类商品房销售面积占比。
> 非住宅 = 总面积 - 住宅面积，再用非住宅/总面积 → **B.20%**

**例7（2023国考）** 2020年防疫类口罩出口额在纺织制品总额中占比。
> 利用2021年数据结合增长率反推2020年，再求占比 → **A.57%**

### 2. A和非A的具体值运用

#### （1）A和非A的值

**例8（2024联考）** 新疆进口金属矿及矿砂、未锻轧铜及铜材、煤炭合计占外贸进口值的比重。
> 部分之和/进口总值 → **C.43%**

**例9（2024江苏事业单位）** 2023年江苏公有制经济增加值。
> 公有制 = GDP - 非公有制，已知非公有制占比75.3%，则公有制占比24.7% → **A.31670.9亿元**

**例10（2024全国事业单位联考）** 除水力发电以外的其他类型总发电量最高的月份。
> 其他 = 总发电量 - 水力发电量，逐月比较 → **A.6月**

#### （2）A和非A的增长量

**例11（2024联考）** 截至2022年12月我国城镇网民规模同比增速。
> 已知总网民增长量，农村网民增长量，利用A+非A=整体反推城镇增长量 → **B.1.6%**

#### （3）A和非A的增长率、混合增长率

**例12（2021联考）** 东部地区除前5名省市以外的互联网业务收入增速。
> 利用整体增长率（东部14.8%）和前5名部分增长率（15.1%）及占比，通过混合增长率反推 → **A.增长9%**

**例13（2020四川下）** 第三产业就业人员占全国就业人员比例最大的年份。
> 第三产业占比 = 第三产业/（第一产业+第二产业+第三产业），或观察变化趋势 → **D.2016年**

#### （4）A和非A的倍数

**例14（2024联考）** 截至2022年12月我国20岁以上网民人数是20岁以下网民的多少倍。
> 利用各年龄段占比，20岁以下=14.2%，20岁以上=1-14.2%=85.8%，倍数=85.8%/14.2% → **C.4-5倍之间**

### 3. 变形题：A和非A在乘积增长率的应用

**方法（求非A的增长率）：**
1. A + 非A = 100%，先求非A比重
2. 利用乘积增长率公式（总量 = A×比重）反向推导

**例16（2019全国事业单位联考）** 2018年我国出口跨境B2C电商交易规模同比增长率。
> 利用B2B和B2C构成总出口跨境电商，已知总增长率（需提取）和B2B增长率（需提取），通过A和非A的乘积增长率反推B2C增长率 → **B.17%**

## 四、综合分析（第十四章）

**常见问法：** "不能从上述资料中推出的是"／"能够从上述资料中推出的是"

**解题策略：**
1. 先扫读四个选项，优先计算简单的（现期量、比重、倍数）
2. 复杂的选项（基期量、增长量、综合分析）后处理
3. 注意"不能推出"类题型的选项陷阱：时间、单位、范围、概念偷换

**例1（2024联考）** 关于2023年一季度新疆外贸的综合判断。
> A.基期量计算（12亿基准）；B.单价比较（进口额增速>进口量增速则单价上升）；C.比重求和；D.比重变化比较 → **D（不能推出）**

**例2（2024国考）** 关于2022年S省12315接收诉求的判断。
> 逐一验证，注意环比增量的识别、倍数的比较、比重的计算 → **D（不能推出）**

| 对比维度 | 县级卷 | 乡镇卷 |

### 判断推理（各35题）

**科学推理（2015年首次出现，5题）**：
1. **视力检查**（平面镜成像）：人到镜子距离=2米（视力表距镜3米，需看5米距离，人到镜2米）→ A
2. **电路分析**：开关通断对电流表影响
3. **蹦极运动**：从B至C过程中运动员速度一直减小 → C（重力<弹力）
4. **溶解度曲线**：温度降低，B物质不析出 → D
5. **蜡烛浮力**：截掉水上部分后剩余部分重新露出水面 → A（因浮力=重力，截后重力减小，需重新平衡）

**必会经典题**：
1. 三集合容斥公式：|A∪B∪C| = |A|+|B|+|C| - |A∩B|-|A∩C|-|B∩C| + |A∩B∩C|
2. 十字交叉法：浓度/平均分的混合问题
3. 工程问题：工作量=效率×时间，合作效率=效率之和
4. 行程问题：相遇距离=速度和×时间，追及距离=速度差×时间
5. 不定方程：用整除/奇偶性缩小范围

### 资料分析

**常用公式**：
| 公式 | 适用场景 |
|------|----------|
| 增长率 = (现期-基期)/基期 | 核心指标 |
| 增长量 = 现期 - 基期 = 基期×增长率 | 计算增长 |
| 比重 = 部分/整体 | 结构分析 |
| 倍数 = A/B | 比较 |
| 平均增长率（间隔）: r = r1 + r2 + r1×r2 | 多年增长 |
| 拉动增长率 = 部分增量/整体基期 | 贡献度 |

**速算技巧**：
- 截位直除法（首数法）
- 有效数字法（四舍五入取前2-3位）
- 特征数字法（1/3≈33.3%, 1/7≈14.3%）
- 估算法（近似比较）
- 速算减法（尾数法）

**资料分析数据特征**：
- 广东卷偏重广东本地经济数据（GDP、民营经济、投资、交通）
- 2014-2015年有大量纯文字材料
- 2016-2017年图表材料增加（需读图）
- 综合判断题（"下列说法正确的是/不正确的是"）每篇至少1题

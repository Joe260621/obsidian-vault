## 2004年上半年

### 数量关系

**数字推理**（5题，每题1分）：
- 题型特征：以递推数列、幂次数列、组合数列为主
- 代表题：
  - 题1：-1, 0, 27, ( ) — 选项：A.64 B.91 C.256 D.512 — 考察幂次规律（-1^3=-1, 0^3=0, 3^3=27, 4^3=64）
  - 题2：1, 2, 0, 3, -1, ( ) — 间隔数列规律
  - 题3：7, 8, 9, 14, 16, 18, 21, 24, ( ) — 三个数字一组，组内等差
  - 题4：0, 0, 1, 4, ( ), 26 — 立方减指数规律（n^3-n^2）
  - 题5：3, -1, 5, 1, ( ) — 间隔递推

**数学运算**（10题，每题1分）：
- **计算问题**：题6：19961997×19971996－19961996×19971997，考察因式分解简化计算
- **行程问题**：题7（圆形跑道，前面7人后面7人=8人）；题10（相遇问题）；题14（门牌号求和）

- **平均数问题**：题9：赵钱孙李成绩，通过方程组求赵的分数

- **淘汰赛**：题12：1044人淘汰赛，决出冠军需1043场（每场淘汰1人，最后剩1人）

### 言语理解与表达

**命题趋势**：
1. **选词填空**（每年稳定10题）：核心为近义词辨析和语境搭配，语素差异是关键
2. **语句表达**（2004-2007共10题，2008删除）：重点为病句辨析（搭配不当、主语残缺、否定误用、歧义）

**高频病句类型**：
- 一面与两面不搭配（"能不能"与"关键在于"）
- 主语残缺（"通过...使..."结构）
- 否定词误用（"防止...不再"）
- 歧义（状语/定语位置不当）
- 语意重复（"至少...以上"、"目前...当前"、"空前...从未"）

**代表篇目知识点**：
- 人文精神与科学精神（2004上）、报复效应（2004下）、电浆电视（2005上）、模糊语言（2005下）、禽流感与达菲（2006）、东西方思维差异（2007）、文化传承与现代化（2008）

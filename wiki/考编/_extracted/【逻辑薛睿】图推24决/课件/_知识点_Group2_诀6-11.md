# 第六诀：均有圆

**口诀**：均有圆整体先看，圆内圆外要分辨，只看曲线交切点，去掉圆把直线算。

**核心规律**：当所有图形都包含圆形（或弧形）时，以圆为参照物进行分析。先观察圆本身的属性（圆与圆是否相交、相切），再看圆内和圆外的元素，最后考虑去掉圆之后剩余直线的规律。

# 第七诀：对称性图形

**口诀**：万物皆可考对称，轴与中心要辨明，数量方向依次数，重叠方式最后评。

**核心规律**：对称性是图形推理的万能考点。解题路径：先判断是轴对称还是中心对称 → 数对称轴数量 → 看对称轴方向 → 检查对称轴之间或与图形重叠方式。

**基础概念**：
- **轴对称**：一个图形，沿着一条直线对折后，直线两边的部分能够完全重合。
- **中心对称**：一个图形绕着某一个点旋转180°后，能和原来的图形完全重合。
- **轴对称+中心对称**：既是轴对称图形（具备至少一条对称轴）又是中心对称的平面图形。

**风车类图形特例**：
- **奇数角风车类图形**：仅轴对称，不中心对称
- **偶数角风车类图形**：既轴对称，又中心对称
- **"不对称单叶"风车型图形**：奇数角不对称，偶数角仅中心对称

# 第八诀：笑脸类

**口诀**：笑脸先看对称轴，曲直部分心中有，鼻嘴眼型莫疏漏，分数到手~

**核心规律**：笑脸类图形以圆形为基本轮廓，通过简化线条或几何形状表现眼睛与嘴巴。解题路径：先看整体对称性 → 再分析曲直性 → 数部分数 → 查看子图形（眼、鼻、嘴）的具体样式 → 数面数量。

# 第九诀：两图形分两部分

**核心定义**：所有图形均由两个独立的几何图形构成。二者之间的关系有三种：**相离**（分离）、**相接**（接触但不重叠）、**重叠**（部分重合）。

**口诀**：图形分成两部分，相离单独看样式，数量特征放心间，重叠莫忘重叠面，相交关注相交边。

# 第十一诀：均有封闭多边形外框

**口诀**：封闭外框是题眼，边面同数为关键，相同最大要敏感，框内框外点与线，三角形面莫数偏，曲平直角细分辨。

**核心规律**：所有图形都有一个封闭的多边形外框。解题时观察外框边数、面数、框内交点数、外框交点数、直角数、平行线数、子图形特征、三角形数、最大面/最小面特征等。

**均有多边形外框能蕴含的信息**：
1. 外框边数，面数量，内部直线数
2. 框内交点数，外框交点数
3. 直角数，平行线数量
4. 子图形特征，三角形数，最大面最小面特征

**解题观察点**（6步法）：
1. **外框边数**有明显变化 → 数外框边数
2. 观察**子图形**是否有明显最大面、相同子图形、对称性子图形
3. 图形外框均一致 → 看**框上交点数、框内交点数、框内直线数**
4. 第一个图起手三角形 → 关注**三角形数量**
5. 有直角三角形、矩形、直角梯形 → 关注**直角**
6. 有曲线、有平行线 → 关注**曲线/平行线数量**

# 第9章 数列问题

> 数列主要考等差数列，求和优先用中项，能做可猜。

---

## 一、等差数列核心公式

### 基本概念
- **通项：** aₙ = a₁ + (n-1)d
- **中项：** 奇数项时中项真实存在（a₍ₙ₊₁₎/₂）；偶数项时取中间两项均值
- **求和：** Sₙ = (a₁+aₙ)/2 × n = a中 × n = 平均数 × n

### 高照技巧
1. **求和有中项，优先用中项。** Sₙ = a中 × n，可做可猜
2. 无中项时用 Sₙ = (a₁+aₙ)/2 × n
3. 常考：3项等差、5项等差、7项等差、9项等差
4. a₁, a₂, a₃ 既是等差又是等比 → a₁=a₂=a₃（常数列）
5. 1-10求和：S₁₀ = 55（高频数据，牢记）

## 二、典型例题

### 题型A：中项直接求

**例3（2022江苏）** 9家企业贷款共4500万，成等差数列，排第3获420万，求排第8。
> a₅ = 4500/9 = 500（9项中项）→ d = (500-420)/2 = 40 → a₈ = 500+3×40 = **A.620万元**

**例4（2022联考）** 9团体新创节目成等差，前5总和60，前7总和70，求总数。
> S₅=5×a₃=60 → a₃=12；S₇=7×a₄=70 → a₄=10 → d=-2, a₅=8 → S₉=9×8=**A.72**

### 题型B：不定方程/推理

**例1（2025上海）** 正二十面体骰子，ri个面上标ri（连续自然数），求数字之和可能值。
> n个连续自然数1~n：总和 = Σi² = n(n+1)(2n+1)/6
> n=5时 = 55 ✓, n=6→91（不在选项）→ **A.55**

**例2（2024江苏）** 4人年龄等差，总和142，一人47岁。求最年长者。
> 设 a, a+d, a+2d, a+3d → 2a+3d=71
> 若 a+2d=47 → a=47-2d → 代入得 d=23, a=1
> 年龄：1, 24, 47, 70 → 最长 **B.70岁**

**例11（2023联考）** 若干玻璃杯，一只空杯，其余有珠共45颗。每有珠杯取1颗入空杯，调整位置后与最初一样。
> 原始：0, 1, 2, ..., n-1（等差数列）
> 操作后：n-1, 0, 1, ..., n-2 → 同集合可重排一致
> 总和 = n(n-1)/2 = 45 → n=10 → **D.10**

### 题型C：增长/递减应用题

**例5（2025全国事业单位）** 果园每天最多采10箱，保存成本10元/箱/天，75箱一次性交付。最低保存成本？
> 最优策略：尽量晚采（早期少采，后期多采）
> 第1天5箱，第2-8天各10箱（共75箱）
> 箱天数 = 5×7 + 10×(6+5+4+3+2+1+0) = 35+210 = 245
> 成本 = 245×10 = **B.2450元**

**例7（2022四川）** 二手车16200元/台，维护费第一年1100，每年+400，年收益9100。哪年利润超2万？
> 累计利润(n) = 9100n - 16200 - [1100n + 200n(n-1)]
> = -200n² + 8200n - 16200 > 20000
> n=6时25800 > 20000 → **B.6**

**例9（2023国考）** 工厂从某周开始，每天比前一天多产200件。第三周产量=第一周×2，求日产量首达1万件的天数。
> 第1天a，第k天 a+200(k-1)
> 第一周总和 = 7a+4200；第三周总和 = 7a+23800
> 7a+23800 = 2(7a+4200) → a=2200
> 2200+200(n-1) ≥ 10000 → n ≥ 40 → **D.40**

### 题型D：等比数列/增长率

**例14（2023联考）** 一月100万包，每月等增长率，四月133.1万包。求增长率。
> 100×(1+r)³ = 133.1 → (1+r)³ = 1.331 → r = **A.10%**

**例15（2025江苏）** 3月初租店(月租1万)，装修2月花11万，第3月营业，当月纯利3万，月增10%。何时开始盈利？
> 累计投入：3-4月租金2万+装修11万=13万
> 5月起：+3-1=+2(累-11), 6月+2.3(累-8.7), 7月+2.63(累-6.07), 8月+2.993(累-3.077), 9月+3.392(累+0.315)
> **D.9月** 首月转正

### 题型E：特殊数列

**例13（2024浙江）** 100个零件编号1-100，反复去掉奇数位后重编号，求最后剩下的原始编号。
> 每次保留偶数位 → 每次除以2。最后剩下的是 ≤100的最大2的幂 = **C.64**

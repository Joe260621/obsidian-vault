# 第1章 倍数特性

## 一、整除判定方法

### 1. 口诀法（用于 3、4、5、9）

| 除数 | 判定规则 | 示例 |
|------|----------|------|
| **3 / 9** | 看各位数字之和能否被 3/9 整除 | 12345 → 1+2+3+4+5=15，可被3整除 |
| **2 / 5** | 看末一位能否被 2/5 整除 | 12125 → 末位5，可被5整除 |
| **4 / 25** | 看末两位能否被 4/25 整除 | 12124 → 末两位24，可被4整除 |
| **8 / 125** | 看末三位能否被 8/125 整除 | 12164 → 末三位164，检验能否被8整除 |

### 2. 拆分法（用于 7、11、13 — 无口诀）

> **核心：** 一个数 = 接近且明显能被整除的数 ± 零头，只看零头能否被整除

- 例：623 ÷ 7 → 把623拆成 7的倍数 ± 零头，只看零头能否被7整除

### 3. 因式分解法（用于 6、12、18、24 等复杂倍数）

> **核心：** 分解成两个**互质**的数，同时满足能被这两个数整除

- 互质：两数没有公约数
- 例：24 = 3 × 8（3和8互质）→ 需同时满足被3和被8整除

## 二、倍数特性三大题型

### 题型一：余数型（均分思想 / 多退少补）

> **核心公式：** 若 总数 = ax + b，则 **(总数 − b)** 能被 a 整除（a、x 均为整数）

**题型特征：** 题干出现"均分"、"多几个"、"少几个"

- 补例2：一堆苹果，每人分10个还剩3个 → 总数−3 是10的倍数
  - A.117 B.120 C.123 D.126 → 代入：123−3=120✓
- 补例3：一堆苹果，每人分10个还缺3个 → 总数+3 是10的倍数
  - A.117 B.120 C.123 D.126 → 代入：117+3=120✓

### 题型二：比例型

> **核心：** 问题和分子、分母的关系

**题干特征：** 分数、比例、百分数、倍数  
**对象特征：** 描述对象为不可分割的整体（人、车、年龄等，整数才有意义）

**四种表达方式（等价）：**

| 表达 | 示例 |
|------|------|
| 分数 | 男员工是女员工的 3/5 |
| 比例 | 男员工与女员工之比 3:5 |
| 百分数 | 男员工是女员工的 60% |
| 倍数 | 男员工是女员工的 0.6 倍 |

**基础结论（男:女 = 7:3）：**

| 问题 | 答案 |
|------|------|
| 男生人数是_____的倍数 | **7** |
| 女生人数是_____的倍数 | **3** |
| 全班人数是_____的倍数 | **10** |
| 男女生人数差是_____的倍数 | **4** |

**三种考法：**
1. **考法一：** 直接给比例，问各部分最小整数解
2. **考法二：** 比例变化的陷阱 — 如 "男−2 : 女 = 7 : 3"（注意变化前后的基准）
3. **考法三（A与非A思想）：** 甲 : 其他 = 7 : 3 → 甲是7的倍数，总人数是10的倍数，其他是3的倍数

## 三、真题例题

### 例1（2025联考）— 比例统一

> 玉米:大豆 = 7:2，大豆:稻谷 = 6:1 → 统一大豆比例 → 玉米:大豆:稻谷 = 21:6:1
> 总面积 61600 ÷ 28 = 2200 × 21 = **46200** ✓A

### 例2（2025联考）— 最优购买

### 例3（2025江苏）— 倍数+选项技巧

> 门票90元/人，教师免票、大学生半票，索道70元/人。共110人，总费用11750元。

### 例4（2025全国事业单位）— 余数型

> 红染料每天消耗3桶，蓝染料每天消耗2桶，7天后红色剩余是蓝色的3倍。问至少购进多少桶？

### 例5（2025山东事业单位）— 同余问题

> 2025人中，编号被3除余1、且被5除余1的会员有多少人？
> 即编号 ≡ 1 (mod 15)，2025÷15=135 → **135人** ✓A

### 例6（2024联考）— 比例变化

> 开班时 男:女=3:4，退出2男1女后 男:女=2:3，问开班时女孩多少人？
> 设男3x女4x → (3x−2):(4x−1)=2:3 → 3(3x−2)=2(4x−1) → 9x−6=8x−2 → x=4 → 女=**16** ✓D

### 例7（2025国考）— 多部门调配

> 甲乙丙丁共210人，甲比丁多10人。乙调20人到丙后剩余并入丁，甲:丙:丁=3:3:4。问原乙部门多少人？
> 设调整后甲=3k,丙=3k,丁=4k → 3k+3k+4k=210 → k=21 → 调整后甲=63,丙=63,丁=84
> 原甲=63,原丁=63−10=53,原丙=63−20=43 → 原乙=210−63−53−43=**51** ✓C

### 例9（2025江苏）— 倍数表述识别

> 2080家企业，未完成比已完成多0.6倍 → 未完成:已完成 = 1.6:1 = 8:5
> 已完成 = 2080 × 5/13 = **800** ✓B

### 例10（2025广东）— 交叉法（混合增长率）

> 蜜柚+40%，柠檬−20%，总+4%。设蜜柚上月占比为x：
> 40%·x + (−20%)·(1−x) = 4% → 40x − 20 + 20x = 4 → 60x = 24 → x = **40%** ✓C

### 例11（2025山东事业单位）— 面积预算

> 960m² → 1000m²，每m²成本需降低？1000×新单价=960×原单价
> 新单价 = 0.96×原单价 → 降低 **4%** ✓D

### 例12（2025联考）— 三层比例+效率

> 老:中=5:2，中:青=3:4 → 老:中:青=15:6:8，采摘速度比1:2:3
> 青采摘=80斤，青速=3×8=24 → 总产量 = 80/24 × (15×1+6×2+8×3) = 80/24 × 51 = 170 ✓A

## 五、学习方法（3+2 作业法）

- **3遍读题：** 先问题、再题干 → 熟悉倍数特性的表述和特征 → 考场快速识别题型
- **2遍计算：** 选项利用（排除、代入）→ 计算是思路的展开
- **核心态度：** 数量要精做、精刷；该做做、该猜猜

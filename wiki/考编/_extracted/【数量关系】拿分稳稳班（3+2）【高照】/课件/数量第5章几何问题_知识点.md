## 一、勾股定理

**核心：** 出现"平方加和"条件 → 直接想勾股定理 a²+b²=c²

### 特殊勾股数（必背）
| 勾股数 | | |
|--------|------|------|
| (3, 4, 5) | (6, 8, 10) | (5, 12, 13) |
| (7, 24, 25) | 边长放缩皆可 |

### 特殊角三角形三边关系

| 角度 | 比例 | 关键 |
|------|------|------|
| **30°-60°-90°** | 1 : √3 : 2 | 短直角边=斜边一半；长直角边=短直角边×√3 |
| **45°-45°-90°** | 1 : 1 : √2 | 两直角边相等；斜边=直角边×√2 |

> √2≈1.414, √3≈1.732, √5≈2.236

## 二、面积与体积公式

### 平面几何
- 圆周长=2πr，圆面积=πr²
- 扇形弧长 = θ/360 × 2πr（θ为中心角）
- 梯形面积=(上底+下底)×高/2

### 立体几何
- 圆柱体积=πr²h，侧面积=2πrh
- 圆锥体积=1/3 πr²h
- 球体积=4/3 πr³，表面积=4πr²

### 例1（2025联考）— 圆弧长度
> 转角α=60°，半径1km → 弧长=60/360×2π=π/3 → 答案为**π/3 km**（选项折算）

### 例3（2025联考）— 圆柱表面积
> 三层圆柱平台，底100/80/68，高5 → 侧面积+顶面积+底面环形
> 总面积=**A.3740π m²**

## 三、几何最值理论

| 原理 | 结论 |
|------|------|
| 周长一定 | 越接近圆，面积越大 |
| 面积一定 | 越接近圆，周长越小 |
| 四边形周长一定 | **正方形**面积最大 |
| 四边形面积一定 | **正方形**周长最小 |

### 同比例放缩
| 放缩N倍 | 变化 |
|----------|------|
| 角度 | 不变 |
| 周长 | ×N（增长N−1倍） |
| 面积 | ×N²（增长N²−1倍） |
| 体积 | ×N³（增长N³−1倍） |

### 例15（2024联考）— 最值+面积
> 100片1m围栏→5块周长相等的矩形。正方形面积最大，长条最小。
> 5×20m周长/块 → 5×5=25 vs 5×(1×9)=45? → **C.16**

### 例16（2025联考）— 圆锥体积放缩
> 倒圆锥，水高为容器一半 → 体积比=(1/2)³=1/8
> 已灌1瓶=1/8V → 还需**B.7瓶**

### 例17（2025联考）— 圆锥切割
> 距顶面1/3处平行切开 → 上面:下面体积比=1³:(3³−1)=1:26?
> → **C.19/8倍**

## 四、方向角问题

**技巧：** 画图！将方位角转化为几何图形的角。

### 常见方位
- 东南方向 = 南偏东45°
- 正北、正南等直接对应坐标系

### 例8（2025山东）— 等速追击
> m点看n点东南方向，沿东偏南30°以相同速度行驶4海里相遇
> → 构建三角形 → mn=**B.2√3海里**

### 例9（2024广东）— 三船方位
> 甲看乙北偏西30°(6km)，甲看丙正西(6km) → 乙丙距 **D.6km**

## 五、平面几何技巧

### 切割与面积比
- 利用等高三角形面积比=底边比
- 四边形对角线分割面积关系

### 例10（2025四川）— 矩形分割
> 正方形分4矩形，B:C=14:9 → 面积推算 → D面积/A面积=**C.2/7**

### 例12（2025浙江）— 三角形面积+距离
> △AOD=9, △AOB=15, △COD=12, CD=8 → 高=面积×2/CD
> → **D.9km**

### 例14（2024国考）— 最短路径（镜像法）
> 两站相距10km，道路平行距12km → 镜像法求最短距离
> 最短=√((10)²+(24)²)=26 → **D.26km**

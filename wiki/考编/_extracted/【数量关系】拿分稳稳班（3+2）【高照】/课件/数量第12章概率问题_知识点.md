# 第12章 概率问题

> 概率比排列组合简单，猜题思维会更多。8类小题型，做猜结合。

## 一、概率基本公式

**P = 满足条件的情况数 / 总情况数**

### 八大题型分类
1. 给情况求概率
2. 给概率求概率
3. 分类思想：正反向思维
4. 几何概率
5. 跟屁虫问题
6. 比赛类概率
7. 抓阄密码类
8. 骰子问题

## 二、给情况求概率

**方法：** 分子分母分别用排列组合计数

**例3（2025国考）** 7天假期学2门课（各需连学2天），随机选安排，3天休息不相邻的概率。
> 总安排：2门课各占2天=4天学习+3天休息。选2个连续2天的位置（不重叠）。
> 不相邻的休息日：插空法。→ **D.1/10**

**练习3（2024联考）** 6个月饼（3蛋黄），任取3个，恰好1个蛋黄。
> C(3,1)×C(3,2)/C(6,3) = 3×3/20 = **A.9/20**

## 三、给概率求概率

**方法：** 利用独立事件乘法和互斥事件加法

**例5（2025联考）** 农作物一年一熟，歉收0.2，丰产0.3。连续3年丰产是连续3年歉收的多少倍？
> 丰产³/歉收³ = 0.3³/0.2³ = 0.027/0.008 = 3.375 → **C.3~4.5之间**

**例6（2025联考）** 小王每题答对概率=小李×1.5。小王连对2题概率比小李高0.2。
> 设小李p，小王1.5p。小王连对：(1.5p)²，小李：p²
> 2.25p² − p² = 0.2 → 1.25p² = 0.2 → p² = 0.16 → p = 0.4
> 小王前2全对且小李前2全错 = (0.6)²×(1-0.4)² = 0.36×0.36 = **B.0.05~0.08**

**例8（2025联考）** 小张小李至少1人签约=3/4，小王签约=1/3。3人至少1人签约？
> P(小张小李都不) = 1−3/4 = 1/4
> P(3人都不) = 1/4 × (1−1/3) = 1/4 × 2/3 = 1/6
> P(至少1人) = 1−1/6 = **D.5/6**

## 四、几何概率

**公式：** P = 有利区域长度(面积)/总区域长度(面积)

**例9（2025联考）** 正方形瓷砖，8菱形+12等腰直角△。蜘蛛停在白色区域的概率。
> 面积比估算 → **C.35%**（需图确认）

**例10（2024联考）** 直角三角形，一直角边外水管7m，整数边长。斜边上点的概率。
> 勾股数：7²+b²=c² → b=24,c=25（7-24-25是勾股数）
> 概率 = 25/(7+24+25) = 25/56 ≈ 0.446 → **C.0.42~0.50**

**例11（2025浙江）** 100m队伍，小明随机站，妈妈随机喊，≤25m能听到。
> 几何概率：线段上两点距离≤25的概率
> 设x为小明位置，y为妈妈位置。|x-y|≤25的概率
> 正方形中带状区域面积 = 1−(1−0.25)² = 1−0.5625 = 0.4375
> = 7/16 → **C.7/16**

## 五、跟屁虫问题

**方法：** 先放一个主体，再放另一个。求在同一排/列/组的概率。

**例13（2018国考）** 5排40座，小张小李随机入座，同一排的概率。
> 先放小张(任意)，小李小张同排：剩下39座中同排剩7座
> P = 7/39 ≈ 17.9% → **B.高于15%低于20%**

**例14（2021江苏）** 9队分3组，前3名同组的概率。
> 先放第一名(任意)，第二名同组：C(2,1)/8，第三名同组：C(1,1)/7
> P = 2/8 × 1/7 = 2/56 = 1/28 → **B.1/28**

## 六、比赛类概率

**方法：** 枚举比赛进程，考虑提前结束条件。

**例15（2025联考）** 甲乙丙依次射击，直至命中。甲0.3,乙0.5,丙0.8。一轮内乙结束/丙结束？
> 乙结束：甲失×乙中 = 0.7×0.5 = 0.35
> 丙结束：甲失×乙失×丙中 = 0.7×0.5×0.8 = 0.28
> 0.35/0.28 = 1.25 → **B.1.25倍**

## 七、抓阄密码类

**方法：** 计算尝试次数内的总概率。

**例16（2020山东）** 密码末两位是两个不同奇数(1,3,5,7,9)。3次尝试内猜中。
> 总可能：A(5,2)=20。3次内猜中 = 3/20 → **A.3/20**

**例17（2021浙江）** 密码后3位全是奇数且有2个相同。2次内猜中。
> 3位奇数，2相同1不同：先选哪两个相同(3选2)，选哪个奇数(5种)，剩下1位选其他4个奇数
> = 3×5×4 = 60种（but需去重排列）
> 具体：总可能 = 5(选重复的奇数)×4(选不同的奇数)×3(不同奇数放哪个位置) = 60
> 2次内猜中 = 2/60 = 1/30 → 但答案 2/59? → 需核实 **D.2/57**

## 八、条件概率/贝叶斯

**练习8（2021四川）** 甲:乙:丙:丁=4:3:2:1，不合格率2%,3%,4%,5%。抽到不合格，是乙的概率。
> P(乙|不合格) = P(乙且不合格)/P(不合格)
> = (3×3%)/(4×2%+3×3%+2×4%+1×5%)
> = 0.09/(0.08+0.09+0.08+0.05) = 0.09/0.30 = 30% → **A.30%**

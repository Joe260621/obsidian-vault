# 第3章 周期问题

## 一、周期余数

**题型特征：** 出现循环或周期，问"第?个"

**解题三步：**
1. **找周期** → 确定周期的起点和长度
2. **算余数** → 总数 ÷ 周期长度 = N个周期 … 余数(n)
3. **做等价** → 第n个：余几数几（无余数=周期最后一个）

### 例1（2019河北）
> 顺次种植：2蜀桧、3刺柏、5小叶女贞、3大叶黄杨，周期=13。2019÷13=155余4 → 第4个 → **B.刺柏**

## 二、周期相遇

> **核心：** 求各主体周期的最小公倍数(LCM)，即为下次相遇间隔

**注意："每隔N天"=每N+1天**

### 例4（2022联考）
> 两信号灯30秒和36秒 → LCM(30,36)=180秒=3分钟
> 10点第1次 → 第7次需要6个间隔 → 6×3=18分钟 → **C.10:18**

### 例5（2019河北）
> 甲每隔2天=每3天，乙每隔4天=每5天，丙每7天 → LCM(3,5,7)=105天
> 4月10日+105天 → 7月24日? 4月剩20+5月31+6月30=81 → 105−81=24 → **C.7月24日**

## 三、星期日期问题

### 大月小月口诀
> **一三五七八十腊，三十一天永不差。**
> 大月31天：1、3、5、7、8、10、12
> 小月30天：4、6、9、11
> 2月：28天(平年) / 29天(闰年)

### 平闰年判定
> 能被4整除为闰年；整百年需被400整除。
> **过平年星期+1，过闰年星期+2**（52周零1天/2天）

### 星期推断核心
- 每连续7天 → 周一到周日各1天
- 每连续28天 → 周一到周日各4天
- **每个月必然有4周** → 取连续28天分析
- **技巧：** 前=后（余数）

### 关键结论速记
| 跨度 | 星期变化 |
|------|----------|
| 平年(365天) | +1 |
| 闰年(366天) | +2 |
| 大月(31天) | +3 |
| 小月(30天) | +2 |

# 第2章 方程问题

## 一、普通方程（一个未知数）

### 设未知数三大技巧

| 技巧 | 场景 | 示例 |
|------|------|------|
| **设小不设大** | 甲是乙的两倍 | 设乙为x，甲=2x |
| **出现比例设份数** | 甲:乙 = 3:5 | 设甲=3x，乙=5x |
| **设中间量** | 甲:乙=3:1，乙:丙=2:3 | 统一乙→设乙=2x，甲=6x，丙=3x |

### 做题步骤
1. 设未知数
2. 找等量关系
3. 约分

## 二、不定方程：ax + by = M

> **关键：** 当 a 或 b 与 M 有公因子时，考虑**倍数特性**

**求解四法：**
- **奇偶性** — 利用奇偶性排除
- **倍数特性** — 公因子约束
- **尾数法** — 5/0 尾数特征
- **代入排除** — 选项验证

**【引例2】** 7x + 3y = 60，y最大为多少？(x,y正整数且不为0)
- 3y = 60−7x，y为整数 → 60−7x是3的倍数
- 代入选项从大到小 → y=13时x=3✓ → **C.13**

## 三、不定方程组

### 第一类：整数型（主流）

> 未知数一定是整数（人、件数等）

**方法：** 先**消元**转化为不定方程 → 再用奇偶、倍数、尾数、代入求解

### 第二类：非整数型（单价/时间等可为小数）

**方法一：凑系数**（高中思维，不易想到）  
**方法二：赋零法** ⭐ — 赋其中一个未知数为零，快速计算其他未知数
> 赋谁为零？**大的数、不好算的数**

## 四、真题例题

### 例1（2025广东）— 均值混合
> 1快捷窗41单，其余窗均28单，总均29单。设总窗口n → 41 + 28(n−1) = 29n → n = **13** ✓A

### 例2（2025天津）— 多未知数方程
> 甲乙丙销售额共1000万，利润共120万。甲销售额=10×甲利润，甲销售额=2/3丙销售额。
> 乙利润比甲多10万，比丙少10万。求乙销售额÷乙利润。
> 设甲利润x → 乙=x+10, 丙=x+20 → x+(x+10)+(x+20)=120 → x=30
> 甲销售额=300 → 丙=450 → 乙销售额=250 → 250/40=**6.25** ✓B

### 例3（2025联考）— 数量相反
> 1.5吨和2吨车各10辆，两种车数量比相反。设甲a辆1.5吨+ (10−a)辆2吨。
> 乙有(10−a)辆1.5吨+ a辆2吨。乙−甲=1吨 → a=... → 乙=**17吨** ✓B

### 例5（2024山东）— 三元一次
> 6人110万，投资10/20/30万。10万比20万多2人。
> 设20万x人 → 10万x+2人 → 30万6−(2x+2)人 → 方程求解 → **B.3**

### 例6（2023联考）— 二元一次
> 5大+1小=3斛，5小+1大=2斛 → 大=1/2斛，小=1/4斛 → 1斛倒满**4个小杯** ✓C

### 例7（2025联考）— 不定方程+最值
> 甲每人参加ABC三项，乙每人任选2项。A=18,B=16,C=15。
> 设甲x人，总人次=3x+(总人数−x)×2 = 18+16+15=49 → 求x最大值 → **A.11**

### 例8（2025联考）— 比例+不定
> 99人，甲75%=3/4研究员，乙88%=22/25研究员。
> 甲人数是4倍数，乙人数是25倍数→甲=24,乙=75
> 甲非研究员=6，乙非研究员=9 → 甲比乙少3人 → **A.少不到5人**

### 例10（2025浙江）— 分数转化整数
> 2分62.5%=5/8，3分44.4%=4/9。设2分出手8x进5x，3分出手9y进4y。
> 10x+12y+罚球=25，罚球7次 → x=1,y=1,罚球中3 → 3/7≈42.9% → **B.40%-50%**

### 例11（2024联考）— 不定方程组整数型
> 15甲+9乙+4丙+1丁=201，共40件，甲=丁。
> 16甲+9乙+4丙=201 → 倍数特性 → 丙=**D.31**

### 例12（2025天津）— 不定方程
> 15人租6车，每车1-4人。求2人车最多几辆。
> 设1人a,2人b,3人c,4人d，a+b+c+d=6，a+2b+3c+4d=15 → b≤**C.3**

### 例13（2025四川）— 非整数型·赋零法
> 甲+乙+丙=2.3万，2甲+乙−丙=2.2万。求甲+3乙+7丙。
> 赋丙=0 → 甲+乙=2.3, 2甲+乙=2.2 → 甲=−0.1不合 → 赋乙=0
> → 甲+丙=2.3, 2甲−丙=2.2 → 甲=1.5,丙=0.8 → 1.5+0+5.6=**B.7.1**

### 例14（2023湖北）— 非整数型·凑系数
> 1苹+3草+5蓝=260，1苹+4草+7蓝=320。求1苹+1草+1蓝。
> ①×2−②: 1苹+2草+3蓝=200 → ②−①: 1草+2蓝=60
> → 1苹+1草+1蓝 = (1苹+2草+3蓝) − (1草+2蓝) = 200−60 = **A.140**

### 例15（2021黑龙江）— 非整数型·赋零法
> 7春+12窗+5狗=200，9春+19窗+5狗=224。求10春+10窗+10狗。
> 减得: 2春+7窗=24。赋窗=0 → 春=12 → 代回: 狗=... → **D.352**

# 第6章 经济利润问题

## 一、基础公式

| 公式 | 说明 |
|------|------|
| 进价(成本) + 利润 = 售价 | 基本等式 |
| 利润率 = 利润 / 成本 | 注意分母是成本不是售价 |
| 进价 × (1+利润率) = 售价 | 由上式推导 |
| 定价 × 折扣 = 售价 | 9折=×0.9 |
| 总利润 = 单利×数量 = 总收入 − 总成本 | |

**切入点选择：** 单件商品→优先设进价；多件商品→优先设数量

### 例1（2025联考）— 盈亏抵消
> 150元卖两张CD，一张盈25%一张亏25%
> 盈利那张成本=150/1.25=120(盈30)，亏损那张成本=150/0.75=200(亏50)
> 净亏20元 → **C.亏损20元**

### 例2（2024广东）— 折扣优惠
> 低于市场价20%，付4000元 → 市场价=4000/0.8=5000 → 优惠**D.1000元**

### 例3（2024江苏）— 混合利润
> 进价100，先加价30%卖，后7折清仓。总利润7410，成本利润率19%。
> 总成本=7410/0.19=39000 → 共购进390个 → **C.105个**

### 例4（2024联考）— 连续降价
> 成本27，定价40，连续两次等幅降价后利润率≤20%
> 设降幅x→40(1−x)²≤27×1.2=32.4 → (1−x)²≤0.81 → 1−x≤0.9 → x≥**C.10%**

### 例5（2023广东事业）— 恢复原价
> 降20%后恢复原价 → 需上调 0.2/0.8=**D.25%**（不是20%！）

## 二、赋值法

**适用条件：** ①给比例求比例 ②三量关系只知其一

**操作：** 对成本或售价赋值；商品数量有比例时优先赋数量

### 例10（2025国考）
> 打8折，原每天40件。打折每天多10件，15天利润=不打折20天利润。
> 赋值定价100 → 成本=**B.64%（即成本/定价=64%）**

## 三、分段计费

**识别：** "……1300元(含)以内……，超出部分……"

**方法：** 先分段→再汇总。设未知数时**设最后一段为x**（不设总长度）

### 例11（2021河北）— 稿费纳税
> 800内免税，800-4000纳10%，4000以上纳15%。纳税620元。
> 前3200×10%=320，剩300/15%=2000 → 总=4000+2000=**D.6000元**

## 四、函数最值（两点式）⭐

**判定：** 单价和销量此消彼长，问何时总收入/总利润最高

**两点式三步：**
1. 设次数，写成两括号相乘：y = (a−x)(b+x)
2. 令y=0，解x₁、x₂
3. 取中点x=(x₁+x₂)/2，此时y最大

### 例12（2020江苏）
> 进价80，售价100，每天120件。每降1元多卖20件。
> 利润=(20−x)(120+20x) → x₁=20,x₂=−6 → x=7 → **C.7元**

### 例13（2022联考）
> 进价40，售价44售300个。每涨1元少10个。
> 利润=(4+x)(300−10x) → x₁=−4,x₂=30 → x=13 → 售价=**D.57元**

### 例15（2024山东）
> 进价8，售价10售100件。提高x元: (2+x)(100−10x)≥350
> 化简 → x²−8x+15≤0 → 3≤x≤5 → 售价13-15 → **B.13-15元**

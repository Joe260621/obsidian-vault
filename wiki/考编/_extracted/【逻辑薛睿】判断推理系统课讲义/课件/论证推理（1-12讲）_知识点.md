# 论证推理（第1-12讲）

> 论证推理的核心不是阅读理解，而是识别考点的底层逻辑结构，切换对应解题思维，匹配正确选项模式。占公考逻辑判断70%左右。

---

## 一、论证推理解题方法论

### 核心认知：考点不等于削弱/支持
**方法：** "削弱""假设""支持"只是问题的表现形式，考点由题干中【论据】与【结论】的推理结构决定。同一道题，正向问就是支持，反向问就是削弱，考点不变。

### 论证推理正确解题步骤
1. **看提问句** —— 问削弱、支持、假设、解释、评价还是选非
2. **找结论引词** —— 因此、所以、据此、由此可见、研究者认为……
3. **找结论** —— 多数在最后一句，少数在第一句
4. **找论据** —— 支持结论的事实、数据、推理
5. **思考推理结构** —— 识别考点，想正确选项可能长什么样
6. **再看选项** —— 优先找心目中的"标准选项"，没有再用排除法

### 读题后的灵魂三问
- 这题在考什么？
- 反驳/支持结论需要什么？（考官出这个正确选项起什么作用？）
- 有没有跟这道题相同逻辑结构的旧题？

### 三要三不要
**三要：**
1. 做对的题多体会底层逻辑结构，横向对比相同考点
2. 做错的题及时复盘，分析错因
3. 优先做近年真题

**三不要：**
1. 不要就题论题死抠字眼
2. 不要凭感觉搞题海战术
3. **不要脑补！** 把日常发散思维收敛为严谨逻辑思维

### 高频考点统计（近10年1013题）
| 题型 | 题量 | 占比 |
|------|------|------|
| 论证推理 | 704 | ~70% |
| 形式逻辑 | 232 | ~23% |
| 分析推理 | 77 | ~7% |

## 二、一句式推理与二句式推理

### 推理结构判定

**一句式推理特征：**
- 论据是实验数据、对比结果、背景信息，**不包含推理**
- 结论包含完整推理
- 论据与结论没有明显的重复主语或重复后半句
- 削弱/支持：**只针对结论的推理**，论据中的细节不参与推理

**二句式推理特征：**
- 论据和结论**均包含推理**
- 论据与结论有重复内容（前一半重复 或 后一半重复）
  - 结构1：论据 A→B，结论 A→C
  - 结构2：论据 A→C，结论 B→C
- 削弱/支持：**必须兼顾论据中的关键词**，论据要参与推理

### 判定目的
区分的关键在于确认：削弱/支持是只针对结论，还是要兼顾论据中的推理链条。存在"断点、概念跳跃"的就是二句式推理。

## 三、二句式推理 —— 补全逻辑

### 推理规则/解题方法
**方法：** 论据和结论之间存在推理链条的缺口，正确选项补上缺失的一环。

**结构1：**
- 论据：A → B
- 结论：A → C
- 正确选项：只要B就C / 只有C才B

**结构2：**
- 论据：A → C
- 结论：B → C
- 正确选项：只要B就A / 只有A才B

### 题型特征
- **题干特征：** 论据和结论各含一个推理，有重复主语或重复结果
- **正确选项特征：** 用"只有……才""只要……就"等逻辑关联词补全推理链条
- **干扰项特征：** 选项只重复题干信息而不补全逻辑；选项涉及无关概念

### 典型题

**例1（2013山东·南方古猿足骨）** 论据：南方古猿有定型的弓形足 → 结论：320万年前就开始像现代人一样用双脚行走
> 推理：弓形足 → 双脚行走，需要补全"弓形足"与"双脚行走"的逻辑关系 → **D. 只有使用双脚行走，才具有定型的弓形足**（结构：论据A→B，结论A→C，补B→C）

**例2（2017国考·火星冰原火山）** 论据：火星表面探测到沸石、硫化物、黏土（冰川下火山喷发产生的物质） → 结论：火星早期覆盖冰原、有过火山活动
> 推理：这些矿物质 → 冰原火山活动，需要补全"物质"与"冰原火山"的唯一对应关系 → **C. 沸石、硫化物和黏土是仅在冰川下的火山活动后才会产生的独特物质**

**例3（2021国考·石英砂陨石坑）** 论据：石英砂有平行直线的断裂结构 → 结论：景观是巨大陨石撞击形成的
> 推理：平行直线断裂结构 → 陨石撞击，补"只有陨石撞击才有这种断裂结构" → **A. 只有经历高速陨石撞击，石英砂才会显示出含有平行直线的断裂结构**

**例4（2025广东·具身智能机器人）** 论据：具身智能机器人强调"大脑"和"小脑"协同，能精准操作且能合理规划任务 → 结论：达芬奇手术机器人不算具身智能
> 推理：达芬奇能精密操作但不能规划任务，补全"规划任务"是必要条件 → **A. 只有具备规划任务能力的机器人才是具身智能机器人 + B. 达芬奇不能做到合理规划任务**（联合前提）

### 常见陷阱
- 选项把论据的条件和结论的关系说反了（如例1中选C"只有弓形足才能双脚行走"就是把因果倒置）
- 选项只重复题干已有的信息，不补充新逻辑链条
- 选项补的是"充分条件"而非必要条件，要看题目具体问什么

## 四、二句式推理 —— 建立联系

### 推理规则/解题方法
**方法：** 论据中的B和结论中的C存在概念跳跃，正确选项在B和C之间建立正向关联。

**结构：**
- 论据：A → B
- 结论：A → C
- 正确选项：B越多C就越多 / B带来的作用、影响因素会导致C

### 题型特征
- **题干特征：** 论据说A导致了B（或A与B有关），结论说A导致了C（或A与C有关），B和C是不同的概念
- **正确选项特征：** 指出B和C之间的正相关关系，或B的机制/作用路径导致C
- **干扰项特征：** 只讨论A和B的关系，或只讨论A和C的关系，不连接B与C

### 典型题

**例1（2019管综联考·猫狗谁聪明）** 论据：猫大脑皮层神经细胞数量是金毛犬的一半 → 结论：狗比猫更聪明
> 推理：神经细胞数量 → 聪明程度，需建立两者联系 → **C. 动物大脑皮层神经细胞的数量与动物的聪明程度呈正相关**

**例2（2017山东·奇虾体长）** 论据：奇虾口器25厘米、粪便化石10厘米 → 结论：体长可能超过2米
> 推理：口器/粪便大小 → 体长，需建立"部位与整体"的关系 → **D. 寒武纪海洋虾类的牙齿和体长有比较固定的构成比例**

**例3（2020山东·久坐影响记忆力）** 论据：坐的时间长与大脑内侧颞叶缩小相关 → 结论：久坐对记忆力产生影响
> 推理：颞叶缩小 → 记忆力下降，需建立联系 → **C. 大脑内侧颞叶区域包含海马回，与记忆形成有关**

**例4（2022国考·熊蜂衰落）** 论据：密集管理型农业推广，熊蜂衰落（比蜜蜂更严重） → 结论：农田利用方式改变对熊蜂威胁更大
> 推理：农业方式 → 熊蜂更受威胁，需建立"农业方式如何导致熊蜂衰落"的机制 → **D. 密集型农田种植单一植物，熊蜂无法如蜜蜂般远距离飞行而岌岌可危**

**例5（2024国考·蕨类植物基因组）** 论据：蕨类植物基因组庞大 → 结论：这有助于它们长期应对环境变化
> 推理：基因组庞大 → 应对环境变化，需建立机制联系 → **C. 对于无性繁殖的植物，庞大的基因组可增加有益突变概率以适应环境**

### 常见的"建立联系"正确选项模板
1. A带来的一个因素/作用/影响导致了B
2. A是B的重要因素、重要标准、重要组成部分；B的好坏取决于A

## 五、二句式推理 —— 指出不同 / 指出相同

### 推理规则/解题方法
**方法：** 论据用A群体（或情境）的情况推理B群体（或情境）的结论，两个对象/情境之间的相似性（支持）或差异性（削弱）是关键。

**结构：**
- 论据：A → B
- 结论：C → D（或B → C）
- 支持：指出A和C相同（类比成立）
- 削弱：指出A和C不同（类比不成立）

### 题型特征
- **题干特征：** 从一个样本/群体/情境的发现，推广到另一个样本/群体/情境（类比推理）
- **正确选项（支持）特征：** 指出两个对象的本质相同
- **正确选项（削弱）特征：** 指出两个对象的本质不同
- **干扰项特征：** 讨论其中一个对象的其他细节，不涉及两者比较

### 典型题

**例1（2022管综联考·线虫延长寿命）** 论据：基因调整使线虫寿命延长5倍 → 结论：应用于人类可活到500岁
> 削弱：指出线虫与人的不同 → **人类寿命提高幅度不能像线虫那样简单倍增**

**例2（1999.10.17·管理者杂志调查）** 论据：《管理者》杂志读者调查显示60%参与者管理知识水平高 → 结论：社会群众管理知识掌握不错
> 削弱：指出读者样本与一般群众不同 → **D. 《管理者》读者主要是高学历者和实际经营管理者**

**例3（2017国考·亚洲女性网购）** 论据：亚洲女性在网购服饰化妆品决定权88% → 结论：喜爱网购的亚洲女性在家庭中有更大控制权
> 削弱：指出"网购决定权"与"家庭控制权"是不同概念 → **A. 喜爱网购的亚洲女性网购支出只占家庭消费支出的25%**

**例4（2018国考·乐观与健康）** 论据：长期保持乐观与悲观的人在死亡率上没有差异 → 结论：心态乐观与否与健康没关系
> 削弱：死亡率 ≠ 健康，指出不同 → **A. 悲观的人更多患有慢性疾病（虽未致命）**

## 六、因果推理 —— 由因推果（预言家思维）

### 推理规则/解题方法
**核心思维（预言家思维）：** 从当前时间点往未来推理，预测在无意外情况下会发生的"果"。

**推理方向：** 原因（论据） → 结果（结论）

**削弱方式（他因削弱）：** 引入**导致结果不成立**（对结果有负面影响）的其他原因

**支持方式：**
1. 建立联系 —— A带来的因素/作用/影响导致了B
2. 指出A是B的重要因素/标准
3. 排除其他使结果不成立的原因

### 题型特征
- **题干特征：** 论据是一个行为/措施/因素（原因），结论是推测未来可能发生的结果
- **正确选项（削弱）特征：** 引入一个在原因存在的情况下、结果仍然可能不成立的他因
- **正确选项（支持）特征：** 建立原因到结果的作用机制
- **干扰项特征（特有陷阱）：** 引入"其他也能导致结果的正面因素"——这不削弱由因推果（因为由因推果关注的是结果是否成立，而不是原因是否唯一）

### 典型题

**例1（2017国考·牛奶助眠）** 论据：牛奶中色氨酸能穿过血脑屏障制造5-羟色胺（助眠荷尔蒙） → 结论：睡前喝牛奶是促进睡眠的有效方法
> 削弱：引入牛奶中其他成分会降低色氨酸作用 → **D. 牛奶中其他氨基酸会争抢穿过血脑屏障的通道，降低色氨酸穿过能力**

**例2（2021浙江·HOV车道）** 论据：车辆高峰期一人使用浪费道路资源 → 结论：设置HOV（多乘员专用车道）可提高道路使用效率、缓解拥堵
> 削弱：引入HOV可能带来负面效果 → **B. 设置HOV增加了变道难度，容易加剧交通拥堵**

**例3（2020国考·站立听课更专注）** 论据：站立听课的学生比坐着的更专注（对比实验） → 结论：站立使人更专注（由因推果）
> 支持：建立站立→专注的机制 → **A. 站立需要大脑平衡身体，这些适度压力使注意力更集中**

**例4（2020国考·废水注入引发地震）** 论据：2014年后地震频繁，同期开始将废水注入地下 → 结论：废水注入是引发地震的关键原因
> 支持：建立"注水→地震"的机制 → **A. 废水灌入岩石水层降低断层间摩擦力，使之更易滑动诱发地震**

**例5（2019国考·长期应激与慢性疼痛）** 论据：长期处于巨大压力下的人群疼痛症状更易迁延 → 结论：长期应激与疼痛慢性化正相关
> 支持：建立应激→疼痛的生理机制 → **B. 长期应激影响神经内分泌系统，使疼痛抑制系统功能被削弱**

**例6（2020国考·藤蔓防林火）** 论据：美洲热带雨林闪电增多但火灾未增 → 结论：可能与藤蔓植物大量增加有关
> 支持：建立藤蔓→防雷火的机制 → **C. 藤蔓茎干电阻更小，能像避雷针传导闪电**

**例7（2017国考·玛雅文明消亡）** 论据：玛雅文明衰落在公元820年起连续95年干旱 → 结论：气候变化是玛雅文明消亡的原因
> 支持：干旱与农耕文明的关系 → **B. 玛雅社会依赖农作物，干旱导致农产品减少，严重影响生存**

## 七、因果推理 —— 由果推因（刑侦思维）

### 推理规则/解题方法
**核心思维（刑侦思维）：** 从已发生的结果出发，往过去推理，分析导致该结果的原因。

**推理方向：** 结果（论据） → 原因（结论）

**削弱方式（他因削弱）：** 引入**导致结果成立的另一个原因**（与结论中的原因无关的其他因素）

**支持方式：**
1. 排除其他可能导致该结果的原因
2. 建立结论中的原因到结果的证据链

### 题型特征
- **题干特征：** 论据是一个已发生的结果/现象/数据，结论推断是什么原因导致了它
- **正确选项（削弱）特征：** 引入另一个也能导致该结果的因素（他因），且该他因与题干论据中的原因无关
- **干扰项特征（特有陷阱）：** 引入"导致结果不成立的"他因 —— 这对由果推因不构成削弱

### 典型题

**例1（2017国考修改·牛奶助眠由果推因版）** 论据：睡前喝牛奶的人入睡更快（结果） → 结论：牛奶是促进睡眠的原因
> 削弱：引入其他因素导致入睡快 → **A. 喝热牛奶如同洗热水浴，皮肤温度上升加快入睡速度**（他因：热效应而非牛奶成分）

**例2（2025国考·桃树套袋小果）** 论据：6月小果占75%（结果） → 结论：套袋过早导致营养竞争不足是小果的原因
> 削弱：引入其他导致营养不足的原因 → **D. 今年桃树长得太快，萌芽肥花后肥都没追施，导致营养供应不足**

**例3（2025国考·文蛤合成红霉素）** 论据：文蛤体内发现红霉素（结果） → 结论：是文蛤体内微生物帮它合成的（原因）
> 削弱：引入另一种合成途径 → **A. 文蛤基因组中具有红霉素合成基因，该基因在其外套膜粘液细胞中表达**（自身就能合成）

**例4（2021浙江修改·HOV缓解拥堵由果推因版）** 论据：推出HOV政策后道路拥堵明显好转（结果） → 结论：是HOV政策带来的成果（原因）
> 削弱：引入其他导致缓解拥堵的因素 → **D. 该市同时也在大力推广鼓励市民乘坐公共交通工具的政策**

**例5（2019辽宁·丁克冲击二孩政策由果推因版）** 论据：生育率下降、选择做丁克的家庭增多（结果） → 结论：丁克观念传播导致人们不愿生孩子（原因）
> 削弱：引入其他导致不愿生二胎的原因 → **C. 城市化快速发展，工作难、房价高、生育成本上涨，是选择不生二胎的主要原因**

### 由因推果 vs 由果推因 对比总结

| 维度 | 由因推果 | 由果推因 |
|------|---------|---------|
| 思维名称 | 预言家思维 | 刑侦思维 |
| 推理方向 | 原因→结果（顺向） | 结果→原因（逆向） |
| 时间指向 | 从过去/现在指向未来 | 从现在指向过去 |
| 核心目的 | 预测可能发生的结果 | 解释已发生结果的原因 |
| 他因削弱 | 引入让结果不成立的他因 | 引入让结果成立的他因（其他原因） |
| 他因支持 | 引入让结果成立的他因 | 排除让结果成立的其他原因 |
| 适用场景 | 科学预测、政策效果预估 | 事故调查、医疗诊断、归因分析 |
| 常见思维缺陷 | 不撞南墙不回头 | 被害妄想症、阴谋论 |

**重要提示：** 由因推果和由果推因的区别**仅在削弱时有不同**，支持和假设的方法相同。

## 八、因果推理 —— 建立联系（支持方向）

### 推理规则/解题方法
**方法：** 题干给出一个因果主张（A导致B），正确选项在A和B之间建立作用机制或排除干扰。

### 正确选项模板
1. A带来的一个因素/作用/影响导致了B（机制路径）
2. A是B的重要因素、重要标准、重要组成部分
3. B的好坏取决于A这个标准

### 典型题

**例1（2020省联考·常喝绿茶有益心血管）** 论据：有喝绿茶习惯者患心脏病和中风风险低20% → 结论：常喝绿茶有益心血管
> 建立机制 → **B. 绿茶中含有的黄酮醇类具有预防血液凝块及血小板成团的作用**

**例2（2019北京·每天吃鸡蛋有利心血管）** 论据：每天吃鸡蛋的人心血管事件风险降低 → 结论：每天吃一个鸡蛋利于心血管健康
> 建立机制 → **C. 食物摄入胆固醇不等于血胆固醇水平，鸡蛋中卵磷脂能阻止胆固醇和脂肪在血管壁沉积**

**例3（2020浙江·准父亲饮食影响后代）** 论据：准父亲肥胖则孩子易肥胖，准父亲饥饿则孩子易患心血管病 → 结论：准父亲的饮食状况会影响后代健康
> 建立机制 → **B. 父亲的营养状况塑造其传递的生殖细胞的信息，影响孩子的生理机能**

**例4（2022国考·羟苯甲酮与珊瑚）** 论据：羟苯甲酮对珊瑚有致命伤害 → 结论：海滨浴场应禁止使用防晒霜
> 建立危害机制 → **C. 羟苯甲酮会破坏、改变珊瑚DNA，降低幼年珊瑚正常发育的几率**

**例5（2018上海·地球氧气大增源自苔藓）** 论据：苔藓类地被植物在4.7亿年前迅速蔓延，之后氧气含量达到今天水平 → 结论：地球氧气大增源自苔藓类植物
> 建立机制 → **C. 苔藓类植物的蔓延增加了沉积岩中有机碳含量，导致空气中氧气含量也增加**

**例6（2023省联考·空气污染与焦虑症）** 论据：空气污染暴露高的区域儿童焦虑症更多 → 结论：空气污染导致焦虑
> 建立机制 → **D. 空气污染会直接影响大脑，细微污染颗粒会对人类精神健康产生影响**

**例7（2024北京·手指操预防跌倒）** 论据：支配手指的神经元在大脑皮层占很大面积 → 结论：老年人做手指操能预防跌倒
> 建立机制 → **C. 特定模式的手指操可以在锻炼大脑的同时提升平衡感，帮助在快跌倒时用手找回平衡**

### 建立联系的"应做-要做"结构
- 论据：做A能导致B → 结论：我们应该做A
- 推理成立需要的隐含前提：**我们需要B / 我们的目标是有B**
- 例："读《红楼梦》有助于了解传统文化 → 年轻人都应该读《红楼梦》"，隐含前提是"年轻人都应该了解传统文化"

## 九、因果推理 —— 指出无关（削弱方向）

### 推理规则/解题方法
**方法：** 题干声称A和B之间存在因果关系，正确选项直接指出A和B之间没有必然联系，即A不是B的决定因素/衡量标准。

### 典型题

**例1（2021浙江·最冷冬天）** 论据：2020年还没到11月就降雪降温 → 结论：2020年冬天是60年来最冷
> 削弱：指出"降雪时间"与"整个冬天平均气温"无关 → **D. 第一场降雪时间与整个冬天平均气温无明显相关**

**例2（2018浙江·综艺节目水准）** 论据：该节目收视率比同类高2% → 结论：该节目达到同类最高水准
> 削弱：指出"收视率"与"节目水准"无必然关联 → **D. 节目水准和收视率并无必然关联**

**例3（2022北京·睡眠短与失智）** 论据：50-70岁睡眠少于6小时失智风险增30% → 结论：中老年人增加睡眠可预防失智
> 削弱：指出失智与睡眠时长无关 → **D. 失智是脑部疾病导致的渐进性认知功能退化，跟睡眠时长无关**

**例4（2025省联考·黑碳暴露与神经系统损伤）** 论据：儿童智力得分与黑碳浓度相关，老年男性MMSE得分与黑碳浓度相关 → 结论：黑碳暴露导致神经系统损伤
> 削弱（指出不同人群的差异） → **A. 相关研究所关注的儿童和老人群体本身血脑屏障较脆弱**（不能推广到所有人）

## 十、引入前提（第8讲）

### 推理规则/解题方法
**方法：** 题干推理存在逻辑缺环，需要补充一个前提使推理成立。与"补全逻辑"的区别在于：引入前提往往不是严格的"只有A才B"的逻辑关联词形式，而是需要补充一个事实性前提或价值判断前提。

### 常见结构
- **论据：** A → B
- **结论：** C → D（或应当做某件事）
- **需要前提：** 连接A与C，或补充目标/价值观前提

### 典型题

**例1（2023国考·深度学习筛查心脏病）** 论据：深度学习技术可以读取视网膜扫描数据识别心脏病风险 → 结论：该技术可彻底改变传统心脏病筛查方式
> 需要前提：视网膜扫描数据与心脏疾病有关 → **A. 视网膜扫描数据反映的微小血管变化是预测心脏疾病较灵敏的指标**

**例2（2025广东·课堂随时提问）** 论据：被鼓励随时提问的班级课堂专注度更高 → 结论：教学中学生应随时提问以提升知识掌握效果
> 需要前提：课堂专注度与知识掌握效果有关 → **B. 课堂专注度是影响知识掌握效果的重要因素**

**例3（2019四川·民生服务与治理能力）** 论据：各区域民生服务水平有了很大提高 → 结论：区域治理能力和成效大幅度提高
> 需要前提：民生服务水平是衡量治理能力的指标 → **B. 民生服务水平是衡量我国区域治理能力和成效的重要指标**

**例4（2023北京·蜘蛛网检测微塑料）** 论据：所有蜘蛛网都被微塑料污染 → 结论：可通过蜘蛛网快速了解区域微塑料污染情况
> 需要前提：蜘蛛网微塑料含量与空气中微塑料浓度正相关 → **C. 蜘蛛网可有效粘着空气中漂浮的微塑料，其含量与空气中的微塑料浓度成正比**

**例5（2022广东·人均收入与满意度）** 论据：人均收入提高与满意度提高正相关 → 结论：政府应大力发展经济增加人均收入
> 需要前提：提高满意度是政府的目标 → **C. 提升人民对生活的满意度是政府工作的主要任务之一**

### 应做结构的前提
- A能导致B → 我们应该做A
- 隐含前提：**我们需要B，B是我们的目标**
- 正确选项就是补上这个"我们需要B"

## 十一、评价题型（第10讲）

### 推理规则/解题方法
**方法：** 评价题的核心是找到题干推理中的关键假设/漏洞，问哪个问题最能帮助判断推理是否成立。正确选项往往是"是则支持、否则削弱"的双向问题。

### 典型题

**例1（2024江苏·小白鼠生物钟基因）** 论据：小白鼠去除生物钟基因后寿命未受显著影响，心血管改善 → 结论：昼夜节律缺失对健康的危害没那么大
> 评价：心血管改善能否代表整体健康？ → **C. 小白鼠去除生物钟基因后的寿命和心血管健康状况能否代表整体健康状况**

**例2（2018上海·酒驾与疲劳驾驶）** 论据：酒驾导致交通事故数量是疲劳驾驶的2倍 → 结论：禁止疲劳驾驶的宣传工作应比禁止酒驾做得更好
> 评价：事故数量与宣传工作是否有直接关系？ → **A. 交通事故数量是否与交通安全方面的宣传工作有直接关系**

## 常见陷阱汇总

### 陷阱1：混淆由因推果与由果推果的他因削弱
- 由因推果：必须引入**让结果不成立**的因素才叫削弱
- 由果推因：必须引入**其他也能导致该结果**的因素才叫削弱
- 两者互换就是错误选项

### 陷阱2：脑补（日常思维发散）
- 不根据题干信息推理，而是用自己的生活经验补充
- 看到"喝牛奶助眠"就脑补牛奶有营养所以能助眠，但实际削弱要找到"牛奶不能助眠的机制"

### 陷阱3：概念偷换
- 论据说"死亡率没差异"结论说"健康没关系"（死亡≠健康）
- 论据说"网购决定权高"结论说"家庭控制权大"（网购决定≠家庭控制）
- 正确选项往往就是指出这两个概念不等同

### 陷阱4：样本与总体的差异（指出不同）
- 从特定样本（杂志读者、实验动物、某地区）推广到一般人群
- 削弱就是指出样本与总体的本质不同

### 陷阱5：建立联系时方向混淆
- 论据：A→B，结论：A→C，正确选项应该是"B与C有关"或"B→C"
- 错误选项通常只谈A与B，或A与C，不连接B和C

### 陷阱6：补全逻辑时条件关系颠倒
- 论据：A→B，结论：A→C
- 正确应为"只有C才B"（B→C），错误选项经常写成"只有B才C"（C→B）

### 陷阱7：不看提问句直接带着预设做题
- "最能支持"和"需要补充的前提"对选项的严谨度要求不同
- "前提"必须是必要条件，"支持"可以是充分条件

### 陷阱8：一句话式和两句式不加区分
- 一句式：只需关注结论的推理
- 二句式：论据中的推理必须参与，正确选项要联系论据中的关键词
